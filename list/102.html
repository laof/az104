Question : 62 Topic: 2

You have an Azure Subscription that contains a storage account named storageacct1234 and two users named User1 and User2.

You assign User1 the roles shown in the following exhibit.
Which two actions can User1 perform? Each correct answer presents a complete solution.
NOTE: Each correct selection is worth one point.

  • A..Assign roles to User2 for storageacct1234.
  • CorrectB..Upload blob data to storageacct1234.
  • C..Modify the firewall of storageacct1234.
  • CorrectD..View blob data in storageacct1234.
  • E..View file shares in storageacct1234.
Suggested Answer:

Correct Answers:

B.. Upload blob data to storageacct1234.
D.. View blob data in storageacct1234.
Explanation: User1 has the Storage Blob Data Contributor role at the storage account level, which allows them to read, write, and delete blob data. Additionally, the Reader role inherited at the resource group level allows User1 to view Azure resource metadata, but not modify it. Therefore, User1 can upload and view blob data.
-----------------

Other Options:

A.. Assign roles to User2 for storageacct1234.
This requires the User Access Administrator or Owner role, which User1 does not have.
C.. Modify the firewall of storageacct1234.
This requires write access to the storage account configuration, which is not granted by either the Reader or Storage Blob Data Contributor roles.
E.. View file shares in storageacct1234.
The Storage Blob Data Contributor role only applies to blob data, not file shares. A separate role like Storage File Data Reader would be needed for that.
