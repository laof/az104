You have an Azure Active Directory (Azure AD) tenant named contoso.com.
You have a CSV file that contains the names and email addresses of 500 external users.
You need to create a guest user account in contoso.com for each of the 500 external users.
Solution: From Azure AD in the Azure portal, you use the Bulk invite users operation.
Does this meet the goal?

  • CorrectA..Yes
  • B..No
Suggested Answer:

Correct Answer:

A.. Yes
Explanation: The Bulk invite users operation in Azure Active Directory allows you to import a CSV file containing external user details (such as names and email addresses) to create guest user accounts in bulk. This method is specifically designed for scenarios like inviting multiple external users efficiently and meets the goal stated in the question.
-----------------

Other Option:

B.. No
This is incorrect because the Bulk invite users feature is the correct and supported method for inviting multiple external users using a CSV file in Azure AD.

Example Process:

Step 1: Prepare a CSV file with the following format:
email,displayName
user1@example.com,User One
user2@example.com,User Two
Step 2: Go to Azure Portal → Azure Active Directory → Users
Step 3: Click Bulk operations → Bulk invite
Step 4: Upload your CSV file
Step 5: Click Submit to start the bulk invitation process
Step 6: Monitor the status under Bulk operation results
-----------------

Notes:

- The invited users will receive an email invitation to join your directory.
- You can customize the invitation message and redirect URL if needed.
- Make sure the CSV file is properly formatted and saved with UTF-8 encoding.
