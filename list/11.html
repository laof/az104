Your company has an Azure Active Directory (Azure AD) tenant named weyland.com that is configured for hybrid coexistence with the on-premises Active
Directory domain.
You have a server named DirSync1 that is configured as a DirSync server.
You create a new user account in the on-premise Active Directory. You now need to replicate the user information to Azure AD immediately.
Solution: You restart the NetLogon service on a domain controller.
Does the solution meet the goal?

  • A..Yes
  • CorrectB..No
Suggested Answer:
B. No
Explanation: Restarting the NetLogon service only affects domain controller authentication services and does not initiate Azure AD synchronization, which requires Azure AD Connect and a sync cmdlet. -----------------
Option A. Yes: Incorrect because NetLogon is unrelated to Azure AD sync and does not replicate user data to Azure AD.
