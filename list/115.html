Question : 75 Topic: 2

HOTSPOT

-
You have an Azure subscription that contains a user named User1 and the resources shown in the following table.
NSG1 is associated to networkinterface1.
User1 has role assignments for NSG1 as shown in the following table.
For each of the following statements, select Yes if the statement is true. Otherwise, select No.
NOTE: Each correct selection is worth one point.

Suggested Answer:

Correct Answers:

User1 can create a storage account in RG1. — No
User1 can modify the DNS settings of networkinterface1. — No
User1 can create an inbound security rule to filter inbound traffic to networkinterface1. — Yes
Explanation: User1 has the Contributor role on NSG1, which allows full management of that resource, including creating inbound security rules. However, User1 does not have Contributor rights on RG1 or networkinterface1, so they cannot create a storage account in RG1 or modify DNS settings on networkinterface1.
-----------------

Explanation of Other Options:

User1 can create a storage account in RG1. — No. The Storage Account Contributor role allows managing storage accounts but not creating them; creation requires Contributor or higher at the resource group level.
User1 can modify the DNS settings of networkinterface1. — No. User1 lacks any role assignment on the network interface or its parent scopes, so they cannot modify its settings.
User1 can create an inbound security rule to filter inbound traffic to networkinterface1. — Yes. User1 has Contributor rights on NSG1, which is associated with the network interface, allowing them to manage its security rules.
-----------------

Additional:

The Storage Account Contributor role allows managing storage account settings such as updating configurations, reading and regenerating access keys, and deleting the storage account. However, it does not grant permissions to:
• Create new storage accounts
• Assign roles or manage access control (RBAC)
• Access blob or queue data via Azure AD (data access requires additional roles like Storage Blob Data Reader or Storage Blob Data Contributor)
• Manage resources outside of storage accounts
