Question : 78 Topic: 2

You have an Azure subscription that contains the resources shown in the following table.

You need to assign User1 the Storage File Data SMB Share Contributor role for share1.
What should you do first?

  • CorrectA..Enable identity-based data access for the file shares in storage1.
  • B..Modify the security profile for the file shares in storage1.
  • C..Select Default to Azure Active Directory authorization in the Azure portal for storage1.
  • D..Configure Access control (IAM) for share1.
Suggested Answer:

Correct Answer:

A. Enable identity-based data access for the file shares in storage1.
Explanation: To assign the Storage File Data SMB Share Contributor role to User1 for share1, you must first enable identity-based access on the storage account. This allows Azure AD-based authentication and authorization for file share access.
-----------------

Explanation of Other Options:

B. Modify the security profile for the file shares in storage1. — Incorrect. This is not required for enabling Azure AD-based access or assigning roles.
C. Select Default to Azure Active Directory authorization in the Azure portal for storage1. — Incorrect. This step is part of the process but comes after enabling identity-based access.
D. Configure Access control (IAM) for share1. — Incorrect. You cannot assign roles until identity-based access is enabled on the storage account.
