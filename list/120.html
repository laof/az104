Question : 80 Topic: 2

HOTSPOT -

You have an Azure AD tenant named adatum.com that contains the groups shown in the following table.
Adatum.com contains the users shown in the following table.
You assign an Azure Active Directory Premium P2 license to Group1 as shown in the following exhibit.
Group2 is NOT directly assigned a license.
For each of the following statements, select Yes if the statement is true. Otherwise, select No.
NOTE: Each correct selection is worth one point.

Suggested Answer:

Question 2-80

1. You can assign User1 the Microsoft Defender for Cloud Apps Discovery license. - No.
2. You can remove the Azure Active Directory Premium P2 license from User1. - No.
3. User2 is assigned the Azure Active Directory Premium P2. - Yes.
-----------------

Explanation:

1. You can assign User1 the Microsoft Defender for Cloud Apps Discovery license. — This is incorrect because assigning this license requires a qualifying plan such as Microsoft 365 E5, Microsoft 365 E5 Security, or EMS E5. The scenario does not indicate that User1 has any of these licenses, so the assignment cannot be assumed.
2. You can remove the Azure Active Directory Premium P2 license from User1. — This is incorrect because User1 receives the license via group-based assignment (Group1). Licenses inherited from groups cannot be removed individually from users; you would need to remove the user from the group or unassign the license from the group.
3. User2 is assigned the Azure Active Directory Premium P2. — This is correct because User2 is a member of Group2, which is a member of Group1. Since Group1 is assigned the license and license inheritance flows through nested groups, User2 inherits the license from Group1.
