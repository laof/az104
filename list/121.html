HOTSPOT

-
You have a hybrid deployment of Azure Active Directory (Azure AD) that contains the users shown in the following table.
You need to modify the JobTitle and UsageLocation attributes for the users.
For which users can you modify the attributes from Azure AD? To answer, select the appropriate options in the answer area.
NOTE: Each correct selection is worth one point.

Suggested Answer:

Question 2-81

JobTitle: User1 and User3 only
UsageLocation: User1 and User3 only
-----------------

Correct Answer: JobTitle: User1 and User3 only | UsageLocation: User1 and User3 only

User1 and User3 only — This is correct because both User1 (cloud-only member) and User3 (guest) are managed directly in Azure AD. Their attributes such as JobTitle and UsageLocation can be modified in Azure AD by an admin. User2 is synced from on-premises, so these attributes must be managed from the on-premises directory and cannot be modified directly in Azure AD.

Explanation of Other Options:

User1 only — Incorrect because it excludes User3, who is also managed in Azure AD and can have attributes modified by an admin.
User1, User2, and User3 — Incorrect because User2 is synced from on-premises and cannot have these attributes modified directly in Azure AD.
