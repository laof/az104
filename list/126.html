HOTSPOT

-
You have an Azure Storage account named storage1 that uses Azure Blob storage and Azure File storage.
You need to use AzCopy to copy data to the blob storage and file storage in storage1.
Which authentication method should you use for each type of storage? To answer, select the appropriate options in the answer area.
NOTE: Each correct selection is worth one point.

Suggested Answer:

Correct Answer:

Blob storage: Azure AD and shared access signatures (SAS) — This is correct because AzCopy supports both Azure AD and SAS token authentication for accessing Azure Blob storage.
File storage: Shared access signatures (SAS) only — This is correct because AzCopy supports SAS tokens for Azure File storage, but Azure AD authentication is not supported for file storage with AzCopy.

Explanation of Other Options:

Azure AD only — Incorrect for both Blob and File storage because while Azure AD is supported for Blob, it is not the only method; and it is not supported for File storage with AzCopy.
Shared access signatures (SAS) only — Incorrect for Blob storage because Azure AD is also supported.
Azure AD and shared access signatures (SAS) — Incorrect for File storage because Azure AD is not supported for file shares with AzCopy.
