HOTSPOT

-
You have an Azure AD tenant that contains a user named External User.
External User authenticates to the tenant by using [email protected].
You need to ensure that External User authenticates to the tenant by using [email protected].
Which two settings should you configure from the Overview blade? To answer, select the appropriate settings in the answer area.
NOTE: Each correct answer is worth one point.

Suggested Answer:

Explanation

Step 1: Modify the email address in identities — This step involves updating the guest user's identity (e.g., changing the email address under the "Identities" section in Azure AD). This is necessary if the user will authenticate using a new email.
Step 2: Click "Reset redemption status" — This clears the previous invitation and allows the guest to re-redeem the invitation using the updated identity. It ensures the new email is used for authentication.
Together, these steps allow the guest user to sign in with the new email address without needing to delete and re-invite them. However, this assumes the identity provider (e.g., Microsoft Account, Google) recognizes the new email and the user can authenticate with it.
