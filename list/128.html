Question : 88 Topic: 2

You have an Azure subscription that contains the resources shown in the following table.

You need to assign Workspace1 a role to allow read, write, and delete operations for the data stored in the containers of storage1.
Which role should you assign?

  • A..Storage Account Contributor
  • B..Contributor
  • CorrectC..Storage Blob Data Contributor
  • D..Reader and Data Access
Suggested Answer:

Correct Answer: C

Storage Blob Data Contributor — This is correct because this role grants read, write, and delete access to blob containers and data within a storage account, which is exactly what Workspace1 needs to interact with the data in storage1.

Explanation of Other Options:

A. Storage Account Contributor — Incorrect because this role allows management of the storage account itself (e.g., configuration), but not access to the data inside containers.
B. Contributor — Incorrect because while it allows management of Azure resources, it does not grant access to the data within storage containers.
D. Reader and Data Access — Incorrect because this role allows read-only access to data and resources, but not write or delete operations.
