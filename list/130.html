Question : 90 Topic: 2

You have an Azure AD tenant that contains the groups shown in the following table.

You purchase Azure Active Directory Premium P2 licenses.
To which groups can you assign a license?

  • A..Group1 only
  • B..Group1 and Group3 only
  • C..Group3 and Group4 only
  • CorrectD..Group1, Group2, and Group3 only
  • E..Group1, Group2, Group3, and Group4
Suggested Answer:

Correct Answer: D

Group1, Group2, and Group3 only — This is correct because Azure AD allows license assignment to Security groups, Mail-enabled security groups, and Microsoft 365 groups — as long as they are enabled. Group4 is a Microsoft 365 group but is disabled, so it cannot be used for license assignment.

Explanation of Other Options:

A. Group1 only — Incorrect because it excludes Group2 and Group3, which are also eligible for license assignment.
B. Group1 and Group3 only — Incorrect because it excludes Group2, which is a mail-enabled security group and also eligible.
E. Group1, Group2, Group3, and Group4 — Incorrect because Group4 is disabled and cannot be used for license assignment.
