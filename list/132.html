Question : 92 Topic: 2

HOTSPOT

-
You have an Azure AD tenant.
You need to modify the Default user role permissions settings for the tenant. The solution must meet the following requirements:
• Standard users must be prevented from creating new service principals.
• Standard users must only be able to use PowerShell or Microsoft Graph to manage their own Azure resources.
Which two settings should you modify? To answer, select the appropriate settings in the answer area.
NOTE: Each correct answer is worth one point.

Suggested Answer:

Correct Answer:

Users can register applications — This is correct because setting this to "No" prevents standard users from creating service principals, which directly addresses the requirement to block app registrations.
Restrict access to Azure AD administration portal — This is correct because enabling this restriction ensures that standard users can only manage their own Azure resources via PowerShell or Microsoft Graph, not through the Azure portal.

Explanation of Other Options:

Users can create security groups — Incorrect because this setting controls group creation, not app registration or portal access.
Guest user access restrictions — Incorrect because the question is about standard users, not guest users.
Linked account connections — Incorrect because this setting is unrelated to service principal creation or resource management.
