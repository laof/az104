Question : 93 Topic: 2

HOTSPOT -

You have an Azure subscription named Sub1 that contains the blob containers shown in the following table.
Sub1 contains two users named User1 and User2. Both users are assigned the Reader role at the Sub1 scope.
You have a condition named Condition1 as shown in the following exhibit.
You have a condition named Condition2 as shown in the following exhibit.
You assign roles to User1 and User2 as shown in the following table.
For each of the following statements, select Yes if the statement is true. Otherwise, select No.
NOTE: Each correct selection is worth one point.

Suggested Answer:

Answer:

User1 can read blob2. | No
User1 can read blob3. | No
User2 can read blob1. | Yes
-----------------

Explanation of Correct Options:

User2 can read blob1. | Yes
User2 has the Storage Blob Data Owner role with Condition 2, which allows access if the action is write or the resource path equals 'cont1'. Since blob1 is in cont1, the condition is satisfied and access is granted.
-----------------

Explanation of Incorrect Options:

User1 can read blob2. | No
User1 has the Storage Blob Data Reader role with Condition 1, which requires the action to match read or the resource path to equal 'cont1'. Blob2 is in cont2, and although the action matches, the condition is not satisfied, so access is denied.
User1 can read blob3. | No
Blob3 is in cont3, which does not match the required resource path 'cont1'. Even though the action matches, the condition is not met, so the role does not apply and access is denied.

Additional:

the user with Read role can view storage account resources, but can't read the data within Azure Storage, such as blobs.
