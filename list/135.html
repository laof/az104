Question : 95 Topic: 2

HOTSPOT -

You purchase a new Azure subscription.
You create an Azure Resource Manager (ARM) template named deploy.json as shown in the following exhibit.
You connect to the subscription and run the following command.
New-AzDeployment –Location westus –TemplateFile “deploy.json”
For each of the following statements, select Yes if the statement is true. Otherwise, select No.
NOTE: Each correct selection is worth one point.

Suggested Answer:

Answer:

1. Three resource groups are created when you run the script. | No
2. A resource group named RGroup5 is created. | No
3. All the resource groups are created in the East US Azure region. | Yes
-----------------

Explanation of Correct Options:

None
No statements are true based on the template and deployment command.
-----------------

Explanation of Incorrect Options:

1. Three resource groups are created when you run the script. | No
The copy loop depends on the length of parameters('obj').list, which is an empty array by default. So it creates 0 resource groups. One additional group is created using the length of parameters('_n4bB_'), which is an empty string, resulting in RG0. Only one group is created.
2. A resource group named RGroup5 is created. | No
No such name is generated. The only group created is RG0.
3. All the resource groups are created in the East US Azure region. | Yes
Although the deployment is run with -Location westus, but the resource groups are created in the Eest Us region, not West US because template don't apply Location parameter, using hardcode.
