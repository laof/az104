Question : 96 Topic: 2

Your on-premises network contains a VPN gateway.

You have an Azure subscription that contains the resources shown in the following table.
You need to ensure that all the traffic from VM1 to storage1 travels across the Microsoft backbone network.
What should you configure?

  • A..Azure AD Application Proxy
  • CorrectB..private endpoints
  • C..a network security group (NSG)
  • D..Azure Peering Service
Suggested Answer:

Correct Answer:

B. private endpoints
Because using a private endpoint for the storage account ensures that traffic from VM1 to storage1 stays within the Microsoft backbone network. A private endpoint assigns a private IP address from your VNet to the storage account, allowing secure and private connectivity.
-----------------

Explanation of Other Options:

A. Azure AD Application Proxy is used to provide secure remote access to on-premises web applications, not for routing traffic between Azure resources.
C. a network security group (NSG) controls inbound and outbound traffic at the subnet or NIC level but does not influence the routing path or ensure use of the Microsoft backbone.
D. Azure Peering Service is designed to optimize connectivity between on-premises networks and Microsoft services over the public internet, not for internal Azure traffic routing.
