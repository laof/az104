Question : 97 Topic: 2

Your on-premises network contains a VPN gateway.

You have an Azure subscription that contains the resources shown in the following table.
You need to ensure that all the traffic from VM1 to storage1 travels across the Microsoft backbone network.
What should you configure?

  • A..Azure AD Application Proxy
  • CorrectB..service endpoints
  • C..a network security group (NSG)
  • D..Azure Firewall
Suggested Answer:

Correct Answer:

B. service endpoints
Service endpoints extend your virtual network private address space to Azure services over the Microsoft backbone network. This ensures that traffic from VM1 to storage1 remains on the Microsoft backbone, improving security and performance.
-----------------

Explanation of Other Options:

A. Azure AD Application Proxy is used to provide secure remote access to on-premises web apps, not for routing traffic between Azure resources.
C. a network security group (NSG) is used to filter network traffic but does not control the routing path or ensure use of the Microsoft backbone.
D. Azure Firewall is a network security service that protects Azure Virtual Network resources but does not ensure traffic uses the Microsoft backbone network.
