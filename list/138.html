Question : 98 Topic: 2

Your on-premises network contains a VPN gateway.

You have an Azure subscription that contains the resources shown in the following table.
You need to ensure that all the traffic from VM1 to storage1 travels across the Microsoft backbone network.
What should you configure?

  • A..Azure Application Gateway
  • CorrectB..service endpoints
  • C..a network security group (NSG)
  • D..Azure Peering Service
Suggested Answer:

Correct Answer:

B. Azure endpoints
Service endpoints (referred to here as "Azure endpoints") allow traffic from VM1 to storage1 to remain on the Microsoft backbone network. They extend your virtual network to Azure services, improving security and performance by avoiding exposure to the public internet.
-----------------

Explanation of Other Options:

A. Azure Application Gateway is a web traffic load balancer that operates at the application layer (OSI layer 7). It is not used for routing traffic between a VM and a storage account.
C. a network security group (NSG) is used to filter traffic but does not control the routing path or ensure use of the Microsoft backbone.
D. Azure Peering Service is designed to enhance connectivity between on-premises networks and Microsoft services over the public internet, not for internal Azure traffic routing.
