You have an Azure subscription that contains the resources shown in the following table.

You need to ensure that data transfers between storage1 and VM1 do NOT traverse the internet
What should you configure for storage1?

  • A..data protection
  • CorrectB..a private endpoint
  • C..Public network access in the Firewalls and virtual networks settings
  • D..a shared access signature (SAS)
Suggested Answer:

Correct Answer:

B. a private endpoint
To ensure that data transfers between VM1 and storage1 do not traverse the internet, you should configure a private endpoint for storage1. A private endpoint assigns a private IP address from your VNet to the storage account, enabling secure, private connectivity over the Microsoft backbone network.
