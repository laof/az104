You have a Microsoft Entra tenant that is linked to the subscriptions shown in the following table.
You have the resource groups shown in the following table.
You assign roles to users as shown in the following table.
For each of the following statements, select Yes if the statement is true. Otherwise, select No.
NOTE: Each correct selection is worth one point.

Suggested Answer:

Correct Answers:

User1 can resize VM1: Yes
User2 can create a new storage account: No
User3 can assign User1 the Owner role for RG3: Yes
User1 has the Contributor role at the MG2 scope. Since Sub3 (which contains RG2 and VM1) is under MG2, User1 can manage resources like resizing VM1, which is permitted by the Contributor role.
User2 is not listed in the roles table and therefore has no assigned role. Without a role like Contributor or Owner, User2 cannot create resources such as a storage account.
User3 has the User Access Administrator role at the Tenant Root Group level. This role allows assigning roles to other users at any scope within the tenant, including assigning the Owner role to User1 for RG3
