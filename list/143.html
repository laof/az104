Question : 103 Topic: 2

Your on-premises network contains a VPN gateway.

You have an Azure subscription that contains the resources shown in the following table.
You need to ensure that all the traffic from VM1 to storage1 travels across the Microsoft backbone network.
What should you configure?

  • A..a network security group (NSG)
  • CorrectB..private endpoints
  • C..Microsoft Entra Application Proxy
  • D..Azure Virtual WAN
Suggested Answer:

Correct Answer:

B. private endpoints
To ensure that traffic from VM1 to storage1 travels across the Microsoft backbone network, you should configure a private endpoint for the storage account. This maps the storage account to a private IP address within Vnet1, ensuring secure and private connectivity without traversing the public internet.
-----------------

Explanation of Other Options:

A. a network security group (NSG) controls traffic flow at the subnet or NIC level but does not affect the routing path or ensure use of the Microsoft backbone.
C. Microsoft Entree Application Proxy (likely a typo for "Entra") is used for secure remote access to on-premises web apps, not for routing Azure VM-to-storage traffic.
D. Azure Virtual WAN is used to connect branch offices and on-premises networks to Azure, but it does not ensure that traffic between VM1 and storage1 stays on the Microsoft backbone.
