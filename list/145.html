Question : 105 Topic: 2

You have an Azure subscription that contains a storage account named storage1.

You need to ensure that the access keys for storage1 rotate automatically.
What should you configure?

  • A..a backup vault
  • B..redundancy for storage1
  • C..lifecycle management for storage1
  • CorrectD..an Azure key vault
  • E..a Recovery Services vault
Suggested Answer:

Correct Answer (with Extended Explanation):

D. an Azure key vault
To ensure that access keys for storage1 rotate automatically, you should configure Azure Key Vault. While Key Vault itself does not natively rotate storage account access keys, it plays a central role in a secure and automated key rotation solution.
To implement automatic rotation, you would typically:
• Store the access keys in Azure Key Vault as secrets.
• Use Azure Automation, Logic Apps, or custom scripts to:
    - Regenerate the storage account access keys.
    - Update the stored values in Key Vault.
    - Automate the rotation process.
This approach minimizes manual effort and enhances security.
-----------------

Explanation of Other Options:

A. a backup vault is used for managing backup data, not for key management or rotation.
B. redundancy for storage1 ensures data durability and availability but does not affect key rotation.
C. lifecycle management for storage1 is used to manage data retention and movement between tiers, not for key rotation.
E. a Recovery Services vault is used for backup and disaster recovery, not for managing or rotating access keys.
