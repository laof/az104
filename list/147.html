Question : 107 Topic: 2

DRAG DROP -

You have a Microsoft Entra tenant.
You need to ensure that when a new Microsoft 365 group is created, the group name is automatically formatted as follows:
Which three actions should you perform in sequence in the Microsoft Entra admin center? To answer, move the appropriate actions from the list of actions to the answer area and arrange them in the correct order.

Suggested Answer:

Correct Answer:

1. Create a group naming policy
2. Set Add prefix to Attribute
3. Set Select type to Department
To automatically format Microsoft 365 group names as <Department><Group name>, you must first create a group naming policy. Then, you configure the policy to add a prefix based on an attribute, and finally, you specify the Department attribute to be used as the prefix.
-----------------

Explanation of Other Options:

Set Add suffix to Attribute is incorrect because the requirement is to add a prefix, not a suffix.
Set Add prefix to String is incorrect because the prefix should be dynamic based on the Department attribute, not a static string.
Customize the company branding is unrelated to group naming policies and affects the sign-in experience, not group name formatting.
