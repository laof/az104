Question : 113 Topic: 2

You have a Microsoft Entra tenant configured as shown in the following exhibit.

The tenant contains the identities shown in the following table.
You purchase a Microsoft Fabric license.
To which identities can you assign the license?

  • A..User1 only
  • CorrectB..User1 and Group1 only
  • C..User1 and Group2 only
  • D..User1, Group1, and Group2
Suggested Answer:

Correct Answer:

B. User1 and Group1 only
-----------------

Explanation of Correct Option:

B. User1 and Group1 only:
Licenses in Microsoft Entra ID can be assigned to individual users and security groups. Group1 is a security group, which supports group-based licensing. User1 is a user account, which can also be directly assigned a license.
-----------------

Explanation of Other Options:

A. User1 only:
This option ignores the valid capability of assigning licenses to security groups like Group1.
C. User1, Group1, and Group2:
Group2 is a Microsoft 365 group, which cannot be used for group-based license assignment in Microsoft Entra ID. Only security groups are supported for this purpose.
