You have Azure subscription that includes data in following locations:

You plan to export data by using Azure import/export job named Export1.
You need to identify the data that can be exported by using Export1.
Which data should you identify?

  • A..DB1
  • CorrectB..container1
  • C..share1
  • D..Table1
Suggested Answer:

B. container1

-----------------
A. DB1 – Incorrect because SQL databases cannot be exported using Azure Import/Export. This service is designed for large-scale data transfer to and from Azure Blob Storage.
C. share1 – Incorrect because Azure Files are not supported by Azure Import/Export. The service only supports Blob Storage.
D. Table1 – Incorrect because Azure Table Storage is not supported by Import/Export. Only Blob data can be exported using this method.
