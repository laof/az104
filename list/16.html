Question : 16 Topic: 1

Your company has three virtual machines (VMs) that are included in an availability set.

You try to resize one of the VMs, which returns an allocation failure message.
It is imperative that the VM is resized.
Which of the following actions should you take?

  • A..You should only stop one of the VMs.
  • B..You should stop two of the VMs.
  • CorrectC..You should stop all three VMs.
  • D..You should remove the necessary VM from the availability set.
Suggested Answer:
C. You should stop all three VMs.
Explanation: Stopping all VMs in an availability set can help resolve allocation failures during resizing by freeing up the required resources in the same fault and update domain.
-----------------
Option A. You should only stop one of the VMs: Insufficient because Azure may still not find enough resources in the required domain.
Option B. You should stop two of the VMs: May not guarantee success; all VMs must be stopped to ensure resource availability.
Option D. You should remove the necessary VM from the availability set: Not recommended as VMs cannot be added or removed from an availability set after creation.
