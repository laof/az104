Question : 6 Topic: 3

You have an Azure subscription that contains the resources in the following table.

Store1 contains a file share named data. Data contains 5,000 files.
You need to synchronize the files in the file share named data to an on-premises server named Server1.
Which three actions should you perform? Each correct answer presents part of the solution.
NOTE: Each correct selection is worth one point.

  • A..Create a container instance
  • CorrectB..Register Server1
  • CorrectC..Install the Azure File Sync agent on Server1
  • D..Download an automation script
  • CorrectE..Create a sync group
Suggested Answer:

Correct Answers

B. Register Server1
C. Install the Azure File Sync agent on Server1
E. Create a sync group
-----------------
C. Install the Azure File Sync agent on Server1 – This is the first step. The agent must be installed on the on-premises server to enable Azure File Sync functionality. At the end of the installation, the server registration process is initiated.
B. Register Server1 – After installing the agent, you register the server with the Azure Storage Sync Service. This step connects the server to Azure for synchronization.
E. Create a sync group – A sync group defines the sync relationship between the Azure file share and the registered server, enabling file synchronization.
A. Create a container instance – Incorrect because container instances are unrelated to Azure File Sync and are used for running containerized applications.
D. Download an automation script – Incorrect because automation scripts are not part of the standard Azure File Sync setup process.
