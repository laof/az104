DRAG DROP -

You have an Azure subscription that contains a storage account.
You have an on-premises server named Server1 that runs Windows Server 2016. Server1 has 2 TB of data.
You need to transfer the data to the storage account by using the Azure Import/Export service.
In which order should you perform the actions? To answer, move all actions from the list of actions to the answer area and arrange them in the correct order.
NOTE: More than one order of answer choices is correct. You will receive credit for any of the correct orders you select.
Select and Place:

Suggested Answer:

Correct Order of Actions for Azure Import/Export

1. Attach an external disk to Server1 and then run waimportexport.exe
2. From the Azure portal, create an import job
3. Detach the external disk from Server1 and then ship the disk to an Azure data center
4. From the Azure portal, update the import job
-----------------
Step 1: Attach an external disk to Server1 and then run waimportexport.exe – This tool prepares the disk by encrypting and copying the data. It also generates journal files needed for the import job.
Step 2: From the Azure portal, create an import job – After preparing the disk, you create the import job and upload the journal files. This registers the job and provides shipping details.
Step 3: Detach the external disk from Server1 and then ship the disk to an Azure data center – The physical disk is sent to Microsoft for data ingestion.
Step 4: From the Azure portal, update the import job – You update the job with shipping and tracking information to complete the process.
