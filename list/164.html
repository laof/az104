DRAG DROP -

You have an Azure subscription named Subscription1.
You create an Azure Storage account named contosostorage, and then you create a file share named data.
Which UNC path should you include in a script that references files from the data file share? To answer, drag the appropriate values to the correct targets. Each value may be used once, more than once or not at all. You may need to drag the split bar between panes or scroll to view content.
NOTE: Each correct selection is worth one point.
Select and Place:

Suggested Answer:

Correct UNC Path

\\contosostorage.file.core.windows.net\data
-----------------
contosostorage – This is the name of the Azure Storage account.
file – This indicates that the resource is a file share, not a blob container.
files.core.windows.net – This is the domain used for accessing Azure file shares.
data – This is the name of the file share created in the storage account.
blob – Incorrect because it refers to blob storage, not file shares.
portal.azure.com – Incorrect because it is the Azure management portal, not used in UNC paths.
subscription1 – Incorrect because subscription names are not part of the UNC path.
