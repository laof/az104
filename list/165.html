HOTSPOT -

You have an Azure subscription that contains an Azure Storage account.
You plan to copy an on-premises virtual machine image to a container named vmimages.
You need to create the container for the planned image.
Which command should you run? To answer, select the appropriate options in the answer area.
NOTE: Each correct selection is worth one point.
Hot Area:

Suggested Answer:

Correct Command to Create the Container

azcopy make 'https://mystorageaccount.blob.core.windows.net/vmimages'
-----------------
azcopy make – Correct because the make command is used to create a container or directory in Azure Storage.
azcopy sync – Incorrect because sync is used to synchronize files between a source and destination, not to create containers.
azcopy copy – Incorrect because copy is used to copy files, not to create containers.
blob – Correct because blob.core.windows.net is the correct domain for accessing blob containers in Azure Storage.
dfs, queue, table, images, file – Incorrect because these are not valid domains or services for blob container creation in this context.
