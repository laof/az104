Suggested Answer:

Question 3-13

1. The virtual machines on the 10.2.9.0/24 subnet will have ________ access to the file shares in the storage account
Answer: C. never
2. Azure Backup will be able to back up the unmanaged hard disks of all the virtual machines in the storage account
Answer: C. never
-----------------
Explanation of correct options:
C. never (for Q1) — Although the subnet is listed in the firewall rules, access to file shares requires either enabling the private endpoint or enabling network routing. Since neither is configured, access is denied.
C. never (for Q2) — Azure Backup is a trusted Microsoft service. Since the checkbox to allow trusted services is not enabled, it cannot access the storage account, even during backup operations.
Explanation of other options:
A. always (for Q1) — Incorrect because access is not granted without a private endpoint or proper routing configuration.
B. during a backup (for Q1) — Incorrect because backup-specific access does not apply to general VM access to file shares.
A. always (for Q2) — Incorrect because Azure Backup does not have persistent access, and trusted services are not allowed.
B. during a backup (for Q2) — Incorrect because even temporary access is blocked when trusted Microsoft services are not allowed.
