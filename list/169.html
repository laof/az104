You have an Azure subscription that contains the storage accounts shown in the following table.

You need to identify which storage account can be converted to zone-redundant storage (ZRS) replication by requesting a live migration from Azure support.
What should you identify?

  • A..storage1
  • CorrectB..storage2
  • C..storage3
  • D..storage4
Suggested Answer:

Question 3-15

Answer: B. Storage2
-----------------
Explanation of correct option:
B. Storage2 — This is a StorageV2 account with Standard performance and LRS replication, which meets the requirements for live migration to ZRS. Azure supports live migration from LRS to ZRS for eligible StorageV2 accounts.
Explanation of other options:
A. Storage1 — Incorrect because it is a general purpose v1 account and uses Premium performance. Live migration to ZRS is not supported for this configuration.
C. Storage3 — Incorrect because it uses Premium performance and RA-GRS replication. Live migration to ZRS is not supported from RA-GRS or for Premium performance tiers.
D. Storage4 — Incorrect because it is a BlobStorage account, not a general-purpose v2 account. BlobStorage accounts are not eligible for live migration to ZRS.

ZRS Support and Conversion Rules

ZRS supports the following Storage Account Types:
- Standard General-purpose v2 Accounts
- Premium File Share Accounts
- Premium Block Blob Accounts
Conversion is only supported for GPv2 and Premium File Share storage accounts (not for Blob Accounts).
To convert from GRS/RA-GRS to ZRS, you must first switch to LRS (direct conversion from RA-GRS is not possible).
