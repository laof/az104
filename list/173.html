Question : 19 Topic: 3

You plan to use the Azure Import/Export service to copy files to a storage account.

Which two files should you create before you prepare the drives for the import job? Each correct answer presents part of the solution.
NOTE: Each correct selection is worth one point.

  • A..an XML manifest file
  • CorrectB..a dataset CSV file
  • C..a JSON configuration file
  • D..a PowerShell PS1 file
  • CorrectE..a driveset CSV file
Suggested Answer:
Answer: B. a dataset CSV file
Answer: E. a driveset CSV file
-----------------
Explanation of correct options:
B. dataset CSV file — This file specifies the list of files and folders to be imported to Azure. It defines the data layout and is required for the import job.
E. driveset CSV file — This file contains metadata about the drives used in the import job, including drive IDs and target storage account information.
Explanation of other options:
A. XML manifest file — Not used in Azure Import/Export; this is not part of the required file set.
C. JSON configuration file — Not required for the import process; Azure Import/Export uses CSV files, not JSON.
D. PowerShell PS1 file — While PowerShell scripts can be used to automate tasks, they are not required files for preparing an import job.
