Question : 20 Topic: 3

You have a Recovery Service vault that you use to test backups. The test backups contain two protected virtual machines.

You need to delete the Recovery Services vault.
What should you do first?

  • A..From the Recovery Service vault, delete the backup data.
  • B..Modify the disaster recovery properties of each virtual machine.
  • C..Modify the locks of each virtual machine.
  • CorrectD..From the Recovery Service vault, stop the backup of each backup item.
Suggested Answer:
Answer: D. From the Recovery Service vault, stop the backup of each backup item.
-----------------
Explanation of correct option:
D. Stop the backup of each backup item — Before you can delete a Recovery Services vault, you must first stop the backup of all protected items. This action unregisters the items and prepares the vault for deletion.
Explanation of other options:
A. Delete the backup data — You cannot delete backup data until the backup is stopped. This step comes after stopping the backup.
B. Modify disaster recovery properties — Not relevant to deleting a Recovery Services vault. This setting is related to replication, not backup.
C. Modify the locks of each virtual machine — VM locks do not affect the ability to delete a Recovery Services vault. This is unrelated to the backup configuration.
