HOTSPOT -

You have an Azure subscription named Subscription1 that contains the resources shown in the following table.
In storage1, you create a blob container named blob1 and a file share named share1.
Which resources can be backed up to Vault1 and Vault2? To answer, select the appropriate options in the answer area.
NOTE: Each correct selection is worth one point.
Hot Area:

Suggested Answer:

Question 3-21

Can use Vault1 for backups:
Answer: VM1 only
Can use Vault2 for backups:
Answer: storage1 only
-----------------
Explanation of correct options:
Vault1 – VM1 only — VM1 is in Central US and Vault1 is also in Central US. Azure Backup requires the Recovery Services vault and the resource to be in the same region, so this is valid.
Vault2 – storage1 only — storage1 is in West US and Vault2 is also in West US. File share backup is supported when both are in the same region.
Explanation of other options:
VM1 and storage1 only — Incorrect for either vault because VM1 and storage1 are in different regions from Vault2 and Vault1 respectively.
VM1 and SQL1 only — Incorrect because SQL1 is in East US, which is a different region from both Vault1 and Vault2.
VM1, storage1, and SQL1 only — Incorrect because SQL1 is in East US and cannot be backed up to either vault.
blob1 and share01 only — blob1 is not listed as a backup-eligible item in this context.
storage01 and SQL01 only — SQL01 is in East US and cannot be backed up to Vault2 in West US.
