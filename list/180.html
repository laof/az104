Question : 26 Topic: 3

You have an Azure subscription that contains an Azure Storage account.

You plan to create an Azure container instance named container1 that will use a Docker image named Image1. Image1 contains a Microsoft SQL Server instance that requires persistent storage.
You need to configure a storage service for Container1.
What should you use?

  • CorrectA..Azure Files
  • B..Azure Blob storage
  • C..Azure Queue storage
  • D..Azure Table storage
Suggested Answer:

Question 3-26 Answer:

Correct Answer: A. Azure Files
-----------------
Explanation of correct option:
Azure Files provides fully managed file shares in the cloud that can be mounted by Azure container instances using the SMB protocol or NFS. It supports persistent storage scenarios like hosting a SQL Server database inside a container, making it the best choice for this use case.
Explanation of other options:
B. Azure Blob storage: Blob storage is optimized for unstructured data like documents, images, and backups. It does not support file system semantics required by SQL Server.
C. Azure Queue storage: Queue storage is used for messaging between application components, not for persistent file storage.
D. Azure Table storage: Table storage is a NoSQL key-value store, not suitable for storing files or databases.
