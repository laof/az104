Question : 28 Topic: 3

You have an Azure subscription named Subscription1.

You have 5 TB of data that you need to transfer to Subscription1.
You plan to use an Azure Import/Export job.
What can you use as the destination of the imported data?

  • A..an Azure Cosmos DB database
  • CorrectB..Azure Blob storage
  • C..Azure Data Lake Store
  • D..the Azure File Sync Storage Sync Service
Suggested Answer:

Question 3-28 Answer:

Correct Answer: B. Azure Blob storage
-----------------
Explanation of correct option:
Azure Import/Export service is designed to transfer large amounts of data to and from Azure Blob storage using hard drives. It supports importing data directly into Blob containers, making it the correct destination for the imported 5 TB of data.
Explanation of other options:
A. Azure Cosmos DB database: Not supported by Azure Import/Export. Cosmos DB requires different data migration tools like Data Migration Tool or Azure Data Factory.
C. Azure Data Lake Store: Azure Import/Export does not support direct import into Data Lake Store. You would need to first import into Blob storage and then move the data.
D. Azure File Sync Storage Sync Service: This service is used to sync on-premises file servers with Azure Files, not for bulk data import via Import/Export.
