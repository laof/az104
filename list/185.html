HOTSPOT -

You have an Azure subscription named Subscription1 that contains the resources shown in the following table:
You plan to configure Azure Backup reports for Vault1.
You are configuring the Diagnostics settings for the AzureBackupReports log.
Which storage accounts and which Log Analytics workspaces can you use for the Azure Backup reports of Vault1? To answer, select the appropriate options in the answer area.
NOTE: Each correct selection is worth one point.
Hot Area:

Suggested Answer:

Storage accounts:

storage3 only

Log Analytics workspaces:

Analytics1, Analytics2, and Analytics3
-----------------

Explanations of right options:

storage3 only: For Azure Backup reports, the storage account must be in the same region as the Recovery Services vault. Vault1 is in West Europe, so only storage3 (in West Europe) is valid.
Analytics1, Analytics2, and Analytics3: The Log Analytics workspace does not need to be in the same region as the Recovery Services vault in most cases. Therefore, all three workspaces (Analytics1 in East US, Analytics2 in West US, Analytics3 in West Europe) are valid options for storing Azure Backup reports for Vault1.
-----------------

Explanation of other options:

storage1 only: storage1 is in East US, not in West Europe, so it cannot be used with Vault1.
storage2 only: storage2 is in West US, not in West Europe, so it is not valid for Vault1.
storage1, storage2, and storage3: Only storage3 is in the same region as Vault1; storage1 and storage2 are in different regions and cannot be used.
Analytics1 only: While Analytics1 is valid, it is not the only valid choice since all three workspaces are supported according to current Azure rules.
Analytics2 only: Analytics2 is valid, but not the only valid workspace.
Analytics3 only: Analytics3 is valid, but current rules allow selection of any workspace, not just Analytics3.
