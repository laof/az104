HOTSPOT -

You have an Azure subscription that contains the storage accounts shown in the following exhibit.
Use the drop-down menus to select the answer choice that completes each statement based on the information presented in the graphic.
NOTE: Each correct selection is worth one point.
Hot Area:

Suggested Answer:

You can create a premium file share in

contoso104 only

You can use the Archive access tier in

contoso101 only or contoso103 only
-----------------

Explanations of right options:

contoso104 only: Premium file shares are supported only in FileStorage accounts. From the list, only contoso104 is of kind FileStorage, so only this account supports premium file shares.
contoso101 only or contoso103 only: The Archive access tier is supported only in general-purpose v2 (StorageV2) and BlobStorage accounts. contoso101 (StorageV2) and contoso103 (BlobStorage) fit this criteria. contoso102 (Storage, general-purpose v1) and contoso104 (FileStorage) do not support the Archive tier.
-----------------

Explanation of other options:

contoso101 only: contoso101 is StorageV2 but does not support premium file shares; only FileStorage does, so this is incorrect.
contoso104 only: This is correct for premium file share, but incorrect for Archive access tier because FileStorage does not support the Archive tier.
contoso101 or contoso104 only: contoso101 is not FileStorage (it is StorageV2), so only contoso104 supports premium file shares.
contoso101, contoso102, or contoso104 only: contoso102 is Storage (v1), and contoso104 is FileStorage, but only contoso104 supports premium file shares, and only StorageV2/BlobStorage support Archive tier.
contoso101, contoso102, contoso103, or contoso104: Only contoso104 supports premium file shares, and only contoso101 and contoso103 support the Archive tier.
contoso101 only or contoso103 only: This is the correct choice for Archive tier since both support it.
contoso101, contoso102, or contoso103 only: This is incorrect for Archive tier because contoso102 (general-purpose v1) does not support Archive.
contoso101, contoso102, and contoso103 only: This is incorrect for Archive tier, as contoso102 (Storage v1) does not support Archive tier.
contoso101, contoso102, contoso103, and contoso104 only: contoso104 is FileStorage, which does not support the Archive tier, and contoso102 is Storage v1, which also does not support the Archive tier.
