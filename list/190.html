Question : 36 Topic: 3

You have an Azure subscription that contains the storage accounts shown in the following table.

You plan to manage the data stored in the accounts by using lifecycle management rules.
To which storage accounts can you apply lifecycle management rules?

  • A..storage1 only
  • B..storage1 and storage2 only
  • C..storage3 and storage4 only
  • CorrectD..storage1, storage2, and storage3 only
  • E..storage1, storage2, storage3, and storage4
Suggested Answer:

D..storage1, storage2, and storage3 only

Explanations of right options:
storage1 (StorageV2), storage2 (BlobStorage), and storage3 (BlockBlobStorage) support lifecycle management rules, which let you automatically manage blob data using defined policies. FileStorage accounts do not support this feature.
-----------------
Explanation of other options:
A..storage1 only – Incorrect, because BlobStorage and BlockBlobStorage accounts also support lifecycle management.
B..storage1 and storage2 only – Incorrect, because BlockBlobStorage is also supported.
C..storage3 and storage4 only – Incorrect, because FileStorage (storage4) does not support lifecycle management rules, and StorageV2 and BlobStorage are omitted.
E..storage1, storage2, storage3, and storage4 – Incorrect, because FileStorage (storage4) does not support lifecycle management rules.

Additional:

The lifecycle management feature is available in all Azure regions for general purpose v2 (GPv2) accounts, blob storage accounts, premium block blobs storage accounts, and Azure Data Lake Storage Gen2 accounts.
