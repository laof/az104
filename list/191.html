Question : 37 Topic: 3

You create an Azure Storage account named contosostorage.

You plan to create a file share named data.
Users need to map a drive to the data file share from home computers that run Windows 10.
Which outbound port should you open between the home computers and the data file share?

  • A..80
  • B..443
  • CorrectC..445
  • D..3389
Suggested Answer:

C..445

Explanations of right options:
Port 445 is used for the SMB protocol required to map network drives to Azure file shares from Windows computers. This port enables communication with Azure Files over the network.
-----------------
Explanation of other options:
A..80 – Incorrect, because port 80 is used for HTTP traffic, not for SMB or file sharing.
B..443 – Incorrect, because port 443 is used for HTTPS traffic, not SMB.
D..3389 – Incorrect, because port 3389 is used for Remote Desktop Protocol (RDP), not for mapping file shares.
