Question : 38 Topic: 3

You have an Azure subscription named Subscription1.

You have 5 TB of data that you need to transfer to Subscription1.
You plan to use an Azure Import/Export job.
What can you use as the destination of the imported data?

  • CorrectA..Azure File Storage
  • B..an Azure Cosmos DB database
  • C..Azure Data Factory
  • D..Azure SQL Database
Suggested Answer:

A..Azure File Storage

Explanations of right options:
Azure Import/Export jobs are designed to transfer large amounts of data to and from Azure Storage accounts, including Azure File Storage and Azure Blob Storage. You can use Azure File Storage as the destination for an import job.
-----------------
Explanation of other options:
B..an Azure Cosmos DB database – Incorrect, because Azure Import/Export does not support Cosmos DB as a direct destination.
C..Azure Data Factory – Incorrect, because Azure Data Factory is a data integration service, not a storage destination.
D..Azure SQL Database – Incorrect, because Azure Import/Export jobs cannot import data directly into SQL databases.
