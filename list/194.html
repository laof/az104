Question : 40 Topic: 3

You have an Azure subscription that contains a storage account named storage1.

You have the devices shown in the following table.
From which devices can you use AzCopy to copy data to storage1?

  • A..Device 1 only
  • CorrectB..Device1, Device2 and Device3
  • C..Device1 and Device2 only
  • D..Device1 and Device3 only
Suggested Answer:

B..Device1, Device2 and Device3

Explanations of right options:
AzCopy is supported on Windows, Linux, and macOS, so you can use AzCopy to copy data to storage1 from all three devices.
-----------------
Explanation of other options:
A..Device1 only – Incorrect, because AzCopy is also available for Linux and macOS.
C..Device1 and Device2 only – Incorrect, because AzCopy is available on macOS as well.
D..Device1 and Device3 only – Incorrect, because AzCopy is available on Linux as well.
