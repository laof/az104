Your company has an Azure Active Directory (Azure AD) subscription.
You want to implement an Azure AD conditional access policy.
The policy must be configured to require members of the Global Administrators group to use Multi-Factor Authentication and an Azure AD-joined device when they connect to Azure AD from untrusted locations.
Solution: You access the multi-factor authentication page to alter the user settings.
Does the solution meet the goal?

  • A..Yes
  • CorrectB..No
Suggested Answer:
Correct Option: B. No
Explanation: Accessing the multi-factor authentication page to alter user settings does not meet the goal because it does not configure a conditional access policy. Conditional access policies are created and managed in the Azure AD Conditional Access section, where you can specify conditions such as requiring Multi-Factor Authentication and an Azure AD-joined device for Global Administrators connecting from untrusted locations.
-----------------
Explanation of Other Options:
A. Yes: This option is incorrect because simply altering user settings on the multi-factor authentication page does not create or configure a conditional access policy, which is necessary to meet the specified requirements.
