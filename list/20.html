DRAG DROP -

You have downloaded an Azure Resource Manager (ARM) template to deploy numerous virtual machines (VMs). The ARM template is based on a current VM, but must be adapted to reference an administrative password.
You need to make sure that the password cannot be stored in plain text.
You are preparing to create the necessary components to achieve your goal.
Which of the following should you create to achieve your goal? Answer by dragging the correct option from the list to the answer area.
Select and Place:

Suggested Answer:
Answer: An Azure Key Vault, An access policy
Explanation: Azure Key Vault is used to securely store secrets like passwords, and an access policy defines who or what can access those secrets, ensuring the password is not stored in plain text in the ARM template.
-----------------
Option: An Azure Storage account: Used for storing data, not secrets.
Option: Azure Active Directory (AD) Identity Protection: Focuses on identity risk management, not secret storage.
Option: An Azure policy: Used for enforcing rules, not for secret management.
Option: A backup policy: Related to backup scheduling, not password security.
