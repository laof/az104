DRAG DROP

-
You have an Azure subscription that contains a virtual machine named VM1.
You need to back up VM1. The solution must ensure that backups are stored across three availability zones in the primary region.
Which three actions should you perform in sequence? To answer, move the appropriate actions from the list of actions to the answer area and arrange them in the correct order.

Suggested Answer:
1. Create a Recovery Services vault.
2. Set Replication to Zone-redundant storage (ZRS).
3. For VM1, create a backup policy and configure the backup.
Explanations:
Create a Recovery Services vault: This is the first step, as the vault stores backup data and backup policies.
Set Replication to Zone-redundant storage (ZRS): ZRS ensures that backup data is stored across three availability zones, which meets the requirement for zone redundancy.
For VM1, create a backup policy and configure the backup: Once the vault and replication settings are ready, configure the backup policy for the VM.
-----------------
Set Replication to Locally-redundant storage (LRS): Incorrect. LRS stores data within a single data center and does not provide zone redundancy.
Configure a replication policy: Incorrect. For Azure VM backup, you set the replication type for the vault, not a separate replication policy.
