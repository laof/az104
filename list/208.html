Question : 54 Topic: 3

You have an Azure subscription named Subscription1.

You have 5 TB of data that you need to transfer to Subscription1.
You plan to use an Azure Import/Export job.
What can you use as the destination of the imported data?

  • A..an Azure Cosmos DB database
  • CorrectB..Azure File Storage
  • C..Azure SQL Database
  • D..a virtual machine
Suggested Answer:
B. Azure File Storage
Explanation: Azure Import/Export jobs are used to transfer large amounts of data to Azure by shipping hard drives to Microsoft. The only supported destinations for imported data are Azure Blob Storage and Azure File Storage. You cannot import data directly into databases (Cosmos DB, SQL Database) or virtual machines using Import/Export jobs.
-----------------
A. an Azure Cosmos DB database: Incorrect. Cosmos DB is not supported as a direct destination for Azure Import/Export jobs.
C. Azure SQL Database: Incorrect. SQL Database is not supported as a direct destination for Azure Import/Export jobs.
D. a virtual machine: Incorrect. You cannot import data directly into a virtual machine using Import/Export jobs.
