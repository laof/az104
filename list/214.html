Question : 60 Topic: 3

You have an Azure subscription named Subscription1.

You have 5 TB of data that you need to transfer to Subscription1.
You plan to use an Azure Import/Export job.
What can you use as the destination of the imported data?

  • CorrectA..Azure Blob Storage
  • B..Azure Data Lake Store
  • C..Azure SQL Database
  • D..a virtual machine
Suggested Answer:
A..Azure Blob Storage
-----------------
B..Azure Data Lake Store
C..Azure SQL Database
D..a virtual machine
Explanation of right option:
Azure Import/Export jobs are designed to import data directly into Azure Blob Storage by shipping supported disks to an Azure datacenter. It does not support direct import to other data services.
Explanation of other options:
B..Azure Data Lake Store: Not supported as a direct destination. You would need to move data from Blob Storage to Data Lake Store using additional tools.
C..Azure SQL Database: Import/Export jobs cannot import data directly into SQL databases; use Data Migration Service or other tools instead.
D..a virtual machine: You cannot import data directly to a VM using Import/Export jobs; only Blob Storage is supported as a destination.
