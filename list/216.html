HOTSPOT

-
You have an Azure subscription that contains the storage accounts shown in the following table.
You need to identify which storage accounts support lifecycle management, and which storage accounts support moving data to the Archive access tier.
Which storage accounts should you use? To answer, select the appropriate options in the answer area.
NOTE: Each correct selection is worth one point.

Suggested Answer:
Lifecycle management: storage1, storage2, and storage3
The Archive access tier: storage2 only
-----------------
Explanation:
Lifecycle management is supported for StorageV2 (storage1), BlobStorage (storage2), and premium BlockBlobStorage (storage3) accounts, so all three support lifecycle management.
The Archive access tier is only supported for accounts configured with LRS, GRS, or RA-GRS, and in this case only storage2 (BlobStorage with RA-GRS) meets that requirement.
BlockBlobStorage (storage3) supports lifecycle management actions but not the archive tier, and StorageV2 (storage1) with GZRS does not support the archive tier either.
