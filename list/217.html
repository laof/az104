Question : 63 Topic: 3

You have an Azure subscription named Subscription1.

You have 5 TB of data that you need to transfer to Subscription1.
You plan to use an Azure Import/Export job.
What can you use as the destination of the imported data?

  • A..an Azure Cosmos DB database
  • B..Azure Data Lake Store
  • CorrectC..Azure Blob storage
  • D..Azure Data Factory
Suggested Answer:
C..Azure Blob storage
-----------------

Explanation of right option:

Azure Import/Export jobs are designed to transfer large amounts of data to or from Azure. The supported destination for imported data is Azure Blob storage. You can import data directly into an Azure storage account as blobs (block blobs or page blobs).

Explanation of other options:

A..an Azure Cosmos DB database: Cosmos DB is a NoSQL database and is not a supported target for Azure Import/Export jobs.
B.. Azure Data Lake Store: Data Lake Store is not supported as a direct destination for Azure Import/Export jobs.
D.. Azure Data Factory: Data Factory is an orchestration service for data integration and ETL, not a storage destination for Import/Export jobs.
