Question : 65 Topic: 3

You have an Azure subscription named Subscription1.

You have 5 TB of data that you need to transfer to Subscription1.
You plan to use an Azure Import/Export job.
What can you use as the destination of the imported data?

  • A..an Azure Cosmos DB database
  • CorrectB..Azure Blob Storage
  • C..Azure SQL Database
  • D..the Azure File Sync Storage Sync Service
Suggested Answer:
B..Azure Blob Storage
-----------------

Explanation of right option:

Azure Blob Storage is the correct destination for imported data using Azure Import/Export jobs. This service is specifically designed to transfer large amounts of data to or from Azure Blob Storage using physical disks.

Explanation of other options:

A..an Azure Cosmos DB database: Cosmos DB is a NoSQL database and is not supported as a direct destination for Import/Export jobs.
C.. Azure SQL Database: SQL Database is a relational database service and is not supported as a direct destination for Import/Export jobs.
D..the Azure File Sync Storage Sync Service: File Sync is used for syncing file shares but is not a direct destination for Import/Export jobs. Only Azure Blob Storage (and Azure Files for export) are supported as import/export destinations.
