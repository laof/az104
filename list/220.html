Question : 66 Topic: 3

You plan to create an Azure Storage account named storage1 that will contain a file share named share1.

You need to ensure that share1 can support SMB Multichannel. The solution must minimize costs.
How should you configure storage?

  • CorrectA..Premium performance with locally-redundant storage (LRS)
  • B..Standard performance with zone-redundant storage (ZRS)
  • C..Premium performance with geo-redundant storage (GRS)
  • D..Standard performance with locally-redundant storage (LRS)
Suggested Answer:
A..Premium performance with locally-redundant storage (LRS)
-----------------

Explanation of right option:

Premium performance with locally-redundant storage (LRS) is the correct configuration because only premium file shares (FileStorage) with LRS or ZRS support SMB Multichannel.

Explanation of other options:

B.. Standard performance with zone-redundant storage (ZRS): Standard file shares (GPv2) with ZRS do not support SMB Multichannel.
C.. Premium performance with geo-redundant storage (GRS): Premium file shares with GRS are not listed as supporting SMB Multichannel.
D.. Standard performance with locally-redundant storage (LRS): Standard file shares (GPv2) with LRS do not support SMB Multichannel.
