Question : 67 Topic: 3

You have an Azure subscription named Subscription1.

You have 5 TB of data that you need to transfer to Subscription1.
You plan to use an Azure Import/Export job.
What can you use as the destination of the imported data?

  • A..Azure Data Lake Store
  • CorrectB..Azure File Storage
  • C..Azure SQL Database
  • D..the Azure File Sync Storage Sync Service
Suggested Answer:
B..Azure File Storage
-----------------

Explanation of right option:

Azure File Storage is a supported destination for Azure Import/Export jobs. You can use Import/Export jobs to transfer data into Azure Files (as well as Azure Blob Storage) by shipping hard drives to Microsoft.

Explanation of other options:

A.. Azure Data Lake Store: Data Lake Store is not supported as a direct destination for Azure Import/Export jobs.
C.. Azure SQL Database: SQL Database is not supported as a destination for Import/Export jobs; it is a relational database and uses different data import mechanisms.
D.. the Azure File Sync Storage Sync Service: Azure File Sync is a service for syncing file shares, not a direct destination for Import/Export jobs.
