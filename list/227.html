Question : 73 Topic: 3

You have an Azure Storage account that contains 5,000 blobs accessed by multiple users.

You need to ensure that the users can view only specific blobs based on blob index tags.
What should you include in the solution?

  • CorrectA..a role assignment condition
  • B..a stored access policy
  • C..just-in-time (JIT) VM access
  • D..a shared access signature (SAS)
Suggested Answer:
A..a role assignment condition

Explanation of right option:

Role assignment condition allows you to use Azure RBAC (role-based access control) with conditions, such as blob index tags, to restrict user access to only those blobs that match certain tags. This is the most secure and granular way to control access based on blob metadata.
-----------------
B..a stored access policy: Stored access policy helps manage shared access signatures (SAS) but does not restrict access by blob index tags.
C..just-in-time (JIT) VM access: JIT VM access is used to secure management ports on virtual machines, unrelated to storage account or blob access.
D..a shared access signature (SAS): SAS provides delegated access to blobs but cannot restrict access by blob index tags alone.
