Question : 74 Topic: 3

You have an Azure Storage account named storage1.

For storage1, you create an encryption scope named Scope1.
Which storage types can you encrypt by using Scope?

  • A..file shares only
  • CorrectB..containers only
  • C..file shares and containers only
  • D..containers and tables only
  • E..file shares, containers, and tables only
  • F..file shares, containers, tables, and queues
Suggested Answer:
B..containers only

Explanation:

Encryption scopes in Azure Storage accounts currently allow you to encrypt blob containers and even scope encryption at the individual blob level. File shares, tables, and queues are not supported by encryption scopes.
-----------------
File shares: Not supported by encryption scopes.
Tables and queues: Not supported by encryption scopes.
