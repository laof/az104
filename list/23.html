DRAG DROP -

Your company has an Azure subscription that includes a number of Azure virtual machines (VMs), which are all part of the same virtual network.
Your company also has an on-premises Hyper-V server that hosts a VM, named VM1, which must be replicated to Azure.
Which of the following objects that must be created to achieve this goal? Answer by dragging the correct option from the list to the answer area.
Select and Place:

Suggested Answer:
Answer:
1. Hyper-V site
2. Azure Recovery Services Vault
3. Replication policy
Explanation: To replicate an on-premises Hyper-V VM to Azure, you must create a Hyper-V site to represent the source environment, a Recovery Services Vault to manage replication, and a Replication policy to define replication behavior.
-----------------
Process Overview:
1. Prepare Azure:
- Create a Recovery Services vault
- storage account, and virtual network.
2. Prepare the Hyper-V Host:
- Install the Azure Site Recovery provider and MARS agent
- register the host.
3. Configure Replication:
- In the vault, select Site Recovery.
- set up a Hyper-V site
- add servers
- configure target replication settings.
4. Enable Replication:
- Select VMs
- apply a replication policy
-enable replication.
-----------------
Option: Storage account: Used in the process but not a required answer object.
Option: Azure Traffic Manager instance: Not related to replication.
Option: Endpoint: Not used in this replication setup.
