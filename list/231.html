Question : 77 Topic: 3

HOTSPOT -

You have an Azure subscription that contains a virtual machine named VM1.
To VM1, you plan to add a 1-TB data disk that meets the following requirements:
• Provides data resiliency in the event of a datacenter outage.
• Provides the lowest latency and the highest performance.
• Ensures that no data loss occurs if a host fails.
You need to recommend which type of storage and host caching to configure for the new data disk.

Suggested Answer:
Storage type: Premium SSD that uses zone-redundant storage (ZRS)
Host caching: None

Explanation of right options:

Premium SSD that uses zone-redundant storage (ZRS) provides data resiliency in the event of a datacenter outage by replicating data across availability zones. It also delivers the lowest latency and highest performance. Host caching: None is recommended for data disks requiring write resiliency, because enabling host caching (especially Read/Write) can lead to potential data loss if the host fails.
-----------------
A. Premium SSD that uses locally-redundant storage (LRS): Only replicates within a single datacenter and does not provide resiliency against datacenter outages.
B. Standard SSD that uses locally-redundant storage (LRS): Lower performance and no cross-zone resiliency.
C. Standard SSD that uses zone-redundant storage (ZRS): While it provides zone redundancy, it does not deliver the highest performance.
D. Host caching: Read-only or Read/Write: Both options can risk data loss if the host fails before data is flushed to storage, violating the requirement to ensure no data loss on host failure.
