Question : 78 Topic: 3

You have an Azure virtual machine named VM1 and an Azure key vault named Vault1.

On VM1, you plan to configure Azure Disk Encryption to use a key encryption key (KEK).
You need to prepare Vault1 for Azure Disk Encryption.
Which two actions should you perform on Vault1? Each correct answer presents part of the solution.
NOTE: Each correct selection is worth one point.

  • A..Select Azure Virtual machines for deployment.
  • CorrectB..Create a new key.
  • C..Create a new secret.
  • D..Configure a key rotation policy.
  • CorrectE..Select Azure Disk Encryption for volume encryption.
Suggested Answer:
B.. Create a new key.
E.. Select Azure Disk Encryption for volume encryption.

Explanation of right options:

Create a new key is required because Azure Disk Encryption with a key encryption key (KEK) needs a key in the key vault.
Select Azure Disk Encryption for volume encryption is the operation in Key Vault that enables the vault for use with Azure Disk Encryption, allowing the VM to use the key for disk encryption.
-----------------
A.. Select Azure Virtual machines for deployment.: This is not an operation performed in Key Vault for Azure Disk Encryption. The relevant setting is enabling Azure Disk Encryption for volume encryption.
C.. Create a new secret.: Secrets are not used for KEK scenarios; this is used for BEK scenarios.
D.. Configure a key rotation policy.: Key rotation is optional and not required for initial Azure Disk Encryption setup.
