HOTSPOT

-
You have an Azure subscription that contains a storage account named storage1.
You need to configure a shared access signature (SAS) to ensure that users can only download blobs securely by name.
Which two settings should you configure? To answer, select the appropriate settings in the answer area.
NOTE: Each correct answer is worth one point.

Suggested Answer:
Allowed resource types: Object (if you want users to access a specific blob by name).
Allowed permissions: Set to "Read" to allow downloading.

Explanation:

The Blob service is required for accessing blobs.
To ensure users can download blobs securely by name, Object must be selected as the allowed resource type, which restricts access to individual blobs.
Read permission allows downloading only, matching the requirement to only download blobs.
-----------------
Allowed resource types: Service or Container would grant broader access than required, allowing users to access more than a specific blob. Other permissions (Write, Delete, etc.) are not needed for read/download-only scenarios and would violate the security requirement.
