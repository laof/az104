Question : 84 Topic: 3

You have an Azure subscription that contains the storage accounts shown in the following table.

Which storage account can be converted to zone-redundant storage (ZRS) replication?

  • A..storage1
  • CorrectB..storage2
  • C..storage3
  • D..storage4
Suggested Answer:
B..storage2
Explanation of right option:
storage2 is a StorageV2 (general purpose v2) account with Locally-redundant storage (LRS). Only StorageV2 accounts with LRS can be converted to Zone-redundant storage (ZRS) replication. This is supported and can be done directly in Azure.
-----------------
A..storage1: Storage1 is a general purpose v1 account, which does not support conversion to ZRS. Only StorageV2 (v2) accounts support this conversion.
C..storage3: Storage3 is a StorageV2 account, but it uses Read-access geo-redundant storage (RA-GRS). You cannot directly convert from RA-GRS to ZRS; you must first convert to LRS before any possible ZRS conversion, making it not directly eligible.
D..storage4: Storage4 is a BlobStorage account, not a StorageV2 account. BlobStorage accounts do not support conversion to ZRS.
