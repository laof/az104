You have an Azure subscription that contains the devices shown in the following table.

On which devices can you install Azure Storage Explorer?

  • A..Device1 only
  • B..Device1 and Device2 only
  • C..Device1 and Device3 only
  • CorrectD..Device1, Device2, and Device3 only
  • E..Device1, Device3, and Device4 only
Suggested Answer:
D..Device1, Device2, and Device3 only
Explanation of right option:
Azure Storage Explorer can be installed on Windows, Ubuntu Linux, and macOS platforms, so you can install it on Device1, Device2, and Device3. Android is not supported for Azure Storage Explorer installation.
-----------------
A..Device1 only: This is incorrect because Azure Storage Explorer also supports Ubuntu Linux and macOS platforms, not just Windows.
B..Device1 and Device2 only: This is incorrect because macOS (Device3) is also supported.
C..Device1 and Device3 only: This is incorrect because Ubuntu Linux (Device2) is also supported.
E..Device1, Device3, and Device4 only: This is incorrect because Device4 (Android) is not a supported platform, and Device2 (Ubuntu Linux) is supported but not included in this option.
