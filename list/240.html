You have an Azure Storage account named storage1.
You need to enable a user named User1 to list and regenerate storage account keys for storage1.
Solution: You assign the Storage Account Key Operator Service Role to User1.
Does this meet the goal?

  • CorrectA..Yes
  • B..No
Suggested Answer:
A..Yes
Explanation of right option:
The Storage Account Key Operator Service Role is designed to allow users to both list and regenerate storage account keys at the storage account level. This role includes the Microsoft.Storage/storageAccounts/listKeys/action and Microsoft.Storage/storageAccounts/regenerateKey/action permissions, which meet the goal of enabling User1 to list and regenerate storage account keys for storage1.
-----------------
B..No: This is incorrect because the Storage Account Key Operator Service Role does provide both list and regenerate permissions for storage account keys, so it meets the requirements stated in the question.
