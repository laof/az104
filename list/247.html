Question : 93 Topic: 3

You have an Azure subscription that contains the storage accounts shown in the following table.

Which storage account can be converted to zone-redundant storage (ZRS) replication?

  • CorrectA..storage1 only
  • B..storage2 only
  • C..storage3 only
  • D..storage2 and storage3
  • E..storage1, storage2, and storage3
Suggested Answer:

Correct answer: A..storage1 only

A..storage1 only — Correct. Only StorageV2 (general purpose v2) accounts with LRS (Locally redundant storage) can be directly converted to ZRS (zone-redundant storage). storage1 meets these criteria.
-----------------
B..storage2 only — Incorrect. storage2 is StorageV2, but its replication is RA-GRS (Read-access geo-redundant storage). You cannot directly convert from RA-GRS to ZRS; only LRS can be converted.
C..storage3 only — Incorrect. storage3 is a BlobStorage (Premium) account, which does not support ZRS replication.
D..storage2 and storage3 — Incorrect. Both storage2 (due to RA-GRS) and storage3 (due to account type) cannot be converted to ZRS.
E..storage1, storage2, and storage3 — Incorrect. Only storage1 supports direct conversion to ZRS; the other two do not meet the requirements.
