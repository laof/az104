Question : 95 Topic: 3

You have an Azure subscription that contains a Standard SKU Azure container registry named ContReg1.

You need to ensure that ContReg1 supports geo-replication.
What should you do first for ContReg1?

  • A..Enable Admin user.
  • B..Add a scope map.
  • C..Add an automation task.
  • D..Create a cache rule.
  • CorrectE..Upgrade the SKU.
Suggested Answer:

Correct answer: E..Upgrade the SKU

E..Upgrade the SKU — Correct. Geo-replication is only supported for Premium SKU Azure Container Registries. To enable geo-replication for ContReg1, you must first upgrade from Standard SKU to Premium SKU.
-----------------
A..Enable Admin user — Incorrect. Enabling the admin user provides admin access credentials but does not affect geo-replication capabilities.
B..Add a scope map — Incorrect. Scope maps are used to define permissions for tokens but do not enable geo-replication.
C..Add an automation task — Incorrect. Automation tasks help with workflows but do not enable geo-replication.
D..Create a cache rule — Incorrect. Cache rules are for caching images from other registries and are unrelated to geo-replication features.
