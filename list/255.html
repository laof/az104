You deploy an Azure Kubernetes Service (AKS) cluster named AKS1.
You need to deploy a YAML file to AKS1.
Solution: From Azure CLI, you run azcopy.
Does this meet the goal?

  • A..Yes
  • CorrectB..No
Suggested Answer:

Answer and Explanation

B..No
Explanation of right option:
azcopy is a command-line utility designed for copying data to and from Azure Storage accounts. It cannot be used to deploy YAML files or resources to an Azure Kubernetes Service (AKS) cluster. The correct tool for deploying YAML files to AKS is kubectl.
