Question : 6 Topic: 4

You download an Azure Resource Manager template based on an existing virtual machine. The template will be used to deploy 100 virtual machines.

You need to modify the template to reference an administrative password. You must prevent the password from being stored in plain text.
What should you create to store the password?

  • CorrectA..an Azure Key Vault and an access policy
  • B..an Azure Storage account and an access policy
  • C..a Recovery Services vault and a backup policy
  • D..Azure Active Directory (AD) Identity Protection and an Azure policy
Suggested Answer:

Answer and Explanations

A.. an Azure Key Vault and an access policy
Explanation of right option: Azure Key Vault is the recommended way to securely store secrets such as passwords, API keys, and certificates. By creating an Azure Key Vault and setting an access policy, you can reference the password securely in your ARM template without storing it in plain text. The VM deployment retrieves the secret at runtime, ensuring security and compliance.
-----------------
B.. an Azure Storage account and an access policy
Incorrect because Azure Storage accounts are for storing data like blobs, files, and tables, not secrets or passwords securely. They do not provide secret management or integration with ARM templates for secure secret retrieval.
C.. a Recovery Services vault and a backup policy
Incorrect because Recovery Services vaults are designed for backup and disaster recovery, not for storing or managing secrets such as administrative passwords.
D.. Azure Active Directory (AD) Identity Protection and an Azure policy
Incorrect because Azure AD Identity Protection is for monitoring and responding to identity-based risks, not for secret storage, and Azure Policy is for governance, not secret management.
