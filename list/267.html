You have an Azure virtual machine named VM1 that runs Windows Server 2016.
You need to create an alert in Azure when more than two error events are logged to the System event log on VM1 within an hour.
Solution: You create an event subscription on VM1. You create an alert in Azure Monitor and specify VM1 as the source
Does this meet the goal?

  • A..Yes
  • CorrectB..No
Suggested Answer:
B..No
Explanation of right option: Creating an event subscription on VM1 and a generic Azure Monitor alert with VM1 as the source does not meet the goal. To trigger an alert when more than two error events are logged to the System event log within an hour, you must configure Azure Monitor with Log Analytics and set up a Log Analytics workspace, connect VM1, and then create a custom log query alert based on event counts.
