You have an Azure virtual machine named VM1. VM1 was deployed by using a custom Azure Resource Manager template named ARM1.json.
You receive a notification that VM1 will be affected by maintenance.
You need to move VM1 to a different host immediately.
Solution: From the Redeploy blade, you click Redeploy.
Does this meet the goal?

  • CorrectA..Yes
  • B..No
Suggested Answer:
A..Yes
Explanation of right option: Using the Redeploy feature from the Redeploy blade in the Azure portal will move the virtual machine to a new physical host in the same Azure region. This action helps to avoid maintenance impact by relocating VM1 to a different host immediately.
-----------------
B..No
Explanation: Not using the Redeploy feature means the VM will remain on the same host and could be affected by the maintenance event. The Redeploy operation is specifically designed for this scenario.
