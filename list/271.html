Question : 19 Topic: 4

You have an Azure subscription that contains a web app named webapp1.

You need to add a custom domain named www.contoso.com to webapp1.
What should you do first?

  • CorrectA..Create a DNS record
  • B..Add a connection string
  • C..Upload a certificate.
  • D..Stop webapp1.
Suggested Answer:
A..Create a DNS record
Explanation of right option: To add a custom domain (www.contoso.com) to an Azure web app, the first step is to create a DNS record (usually a CNAME or A record) that points www.contoso.com to the Azure web app's default domain. This verifies domain ownership and allows Azure to complete the custom domain setup.
-----------------
B..Add a connection string
Explanation: Connection strings are used for app configuration, not for domain setup.
C..Upload a certificate.
Explanation: Uploading a certificate is required only if you want to enable HTTPS for the custom domain, not for mapping the domain itself.
D..Stop webapp1.
Explanation: Stopping the web app is unnecessary for adding a custom domain.
