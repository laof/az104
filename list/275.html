HOTSPOT -

You have an Azure subscription named Subscription1 that contains the quotas shown in the following table.
You deploy virtual machines to Subscription1 as shown in the following table.
You plan to deploy the virtual machines shown in the following table.
For each of the following statements, select Yes if the statement is true. Otherwise, select No.
NOTE: Each correct selection is worth one point.
Hot Area:

Suggested Answer:
You can deploy VM3 to West US. Yes.
You can deploy VM4 to West US. Yes.
You can deploy VM5 to West US. No.
Explanation:
Both VM3 (Standard_B2ms, 1 vCPU, BS Family) and VM4 (Standard_D4s_v3, 4 vCPUs, D Family) can be deployed because there is quota available in their respective vCPU families (0 of 20 used for both BS and D Families in West US).
VM5 (Standard_B16ms, 16 vCPUs, BS Family) cannot be deployed because deploying VM1 (2 vCPUs) + VM3 (1 vCPU) + VM5 (16 vCPUs) would exceed the BS Family quota of 20 vCPUs.
Additional:
Deallocated VMs do not consume vCPU quota; only running or allocated VMs count against the quota.
