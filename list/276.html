HOTSPOT -

You have an Azure subscription that contains an Azure Availability Set named WEBPROD-AS-USE2 as shown in the following exhibit.
You add 14 virtual machines to WEBPROD-AS-USE2.
Use the drop-down menus to select the answer choice that completes each statement based on the information presented in the graphic.
NOTE: Each correct selection is worth one point.
Hot Area:

Suggested Answer:
When Microsoft performs planned maintenance in East US 2, the maximum number of unavailable virtual machines will be 2.
If the server rack in the Azure datacenter that hosts WEBPROD-AS-USE2 experiences a power failure, the maximum number of unavailable virtual machines will be 7.
Explanation of right options:
For planned maintenance, only VMs in one update domain (out of 10) are affected at a time, so with 14 VMs, the maximum per domain is 2 (ceiling(14/10)=2).
For a fault domain (rack) failure, VMs are split across 2 fault domains, so the maximum in one domain is 7 (ceiling(14/2)=7).
-----------------
When Microsoft performs planned maintenance in East US 2, the maximum number of unavailable virtual machines will be 7, 10, or 14: These are incorrect because update domains limit unavailability to those in a single domain, which is at most 2 for 14 VMs and 10 update domains.
If the server rack in the Azure datacenter that hosts WEBPROD-AS-USE2 experiences a power failure, the maximum number of unavailable virtual machines will be 2, 10, or 14: These are incorrect because a single rack (fault domain) failure only affects VMs in that domain, which is at most 7 for 14 VMs and 2 fault domains.
