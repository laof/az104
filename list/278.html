Question : 26 Topic: 4

You have a deployment template named Template1 that is used to deploy 10 Azure web apps.

You need to identify what to deploy before you deploy Template1. The solution must minimize Azure costs.
What should you identify?

  • A..five Azure Application Gateways
  • CorrectB..one App Service plan
  • C..10 App Service plans
  • D..one Azure Traffic Manager
  • E..one Azure Application Gateway
Suggested Answer:
B..one App Service plan
Explanation of right option:
One App Service plan can host multiple Azure web apps, allowing you to deploy all 10 web apps while minimizing Azure costs by sharing resources under a single plan.
-----------------
A..five Azure Application Gateways: Incorrect, as Application Gateways are used for Layer 7 load balancing and are not required for deploying web apps or minimizing costs.
C..10 App Service plans: Incorrect, as creating 10 separate App Service plans would increase costs; you only need one plan to host multiple web apps.
D..one Azure Traffic Manager: Incorrect, as Traffic Manager is used for DNS-based global traffic distribution, not for deploying or hosting web apps.
E..one Azure Application Gateway: Incorrect, as only an App Service plan is required to host web apps, and an Application Gateway is not needed for basic deployment.
