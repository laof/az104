Question : 32 Topic: 4

You plan to create the Azure web apps shown in the following table.

What is the minimum number of App Service plans you should create for the web apps?

  • A..1
  • CorrectB..2
  • C..3
  • D..4
Suggested Answer:
B..2
Explanation of right option: You only need two App Service plans because both WebApp1 (.NET Core 3.1) and WebApp2 (ASP.NET V 4.8) can share a plan that supports Windows-based stacks, while WebApp3 (PHP 7.3) and WebApp4 (Ruby 2.6) can share a plan that supports Linux-based stacks. Windows and Linux apps cannot be hosted on the same App Service plan, but different languages within the same OS can share a plan.
-----------------
A..1: Not correct because Windows and Linux apps cannot be mixed on a single App Service plan.
C..3: Not correct because PHP and Ruby can share a Linux-based plan, so you don't need three plans.
D..4: Not correct because you do not need a separate plan for each app; some can share based on OS compatibility.
