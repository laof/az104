You have an Azure subscription that contains the resources shown in the following table.
VM1 connects to VNET1.
You need to connect VM1 to VNET2.
Solution: You create a new network interface, and then you add the network interface to VM1.
Does this meet the goal?

  • A..Yes
  • CorrectB..No
Suggested Answer:
B..No
Explanation: You cannot add a network interface from a different region to a VM. VM1 is in "West US" and VNET2 is in "East Asia." Network interfaces must be in the same region as the VM and the virtual network. Therefore, you cannot connect VM1 (West US) to VNET2 (East Asia) by creating a new network interface.
