You have an Azure subscription named Subscription1 that contains the resources shown in the following table.

You create virtual machines in Subscription1 as shown in the following table.
You plan to use Vault1 for the backup of as many virtual machines as possible.
Which virtual machines can be backed up to Vault1?

  • A..VM1 only
  • B..VM3 and VMC only
  • C..VM1, VM2, VM3, VMA, VMB, and VMC
  • CorrectD..VM1, VM3, VMA, and VMC only
  • E..VM1 and VM3 only
Suggested Answer:
D. VM1, VM3, VMA, and VMC only
Explanation:
Vault1 is in West Europe, so it can only back up VMs in the same region (West Europe). These VMs are: VM1, VM3, VMA, VMC. VM2 and VMB are in North Europe and cannot be backed up to a vault in West Europe.
Additional:
The following criteria is important for vault backup: the data source (VM) must be in the same region and subscription.
