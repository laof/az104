You have an Azure subscription named Subscription1. Subscription1 contains a resource group named RG1. RG1 contains resources that were deployed by using templates.
You need to view the date and time when the resources were created in RG1.
Solution: From the RG1 blade, you click Automation script.
Does this meet the goal?

  • A..Yes
  • CorrectB..No
Suggested Answer:
B. No
Explanation:
A. Yes: Clicking Automation script in the RG1 blade only shows you the template and parameters used for deployment, not the creation date and time of the resources.
B. No: The Automation script blade does not display resource creation timestamps. To find creation times, you should check the deployment history or Azure Activity Log for RG1.
