You have an Azure subscription named Subscription1.

You deploy a Linux virtual machine named VM1 to Subscription1.
You need to monitor the metrics and the logs of VM1.
What should you use?

  • A..Azure HDInsight
  • CorrectB..Linux Diagnostic Extension (LAD) 3.0
  • C..the AzurePerformanceDiagnostics extension
  • D..Azure Analysis Services
Suggested Answer:
B. Linux Diagnostic Extension (LAD) 3.0
Explanation:
A. Azure HDInsight: HDInsight is an analytics service for big data, not for monitoring VM metrics or logs.
B. Linux Diagnostic Extension (LAD) 3.0: LAD 3.0 is specifically designed to collect metrics and logs from Linux VMs in Azure.
C. the AzurePerformanceDiagnostics extension: This extension is for troubleshooting performance issues but is not typically used for ongoing metrics and log collection.
D. Azure Analysis Services: This is a data analysis platform, not a monitoring tool for VM metrics or logs.
