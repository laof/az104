Question : 49 Topic: 4

You plan to deploy three Azure virtual machines named VM1, VM2, and VM3. The virtual machines will host a web app named App1.

You need to ensure that at least two virtual machines are available if a single Azure datacenter becomes unavailable.
What should you deploy?

  • A..all three virtual machines in a single Availability Zone
  • B..all virtual machines in a single Availability Set
  • CorrectC..each virtual machine in a separate Availability Zone
  • D..each virtual machine in a separate Availability Set
Suggested Answer:
C. each virtual machine in a separate Availability Zone
-----------------
Explanation of correct option:
each virtual machine in a separate Availability Zone: By deploying each VM in a different Availability Zone, the solution ensures that if a single Azure datacenter (which corresponds to a single Availability Zone) becomes unavailable, the other VMs in the remaining zones will continue to operate. This provides the highest level of resilience against datacenter failures.
-----------------
Explanation of other options:
A. all three virtual machines in a single Availability Zone: This would place all VMs in the same datacenter, so if that zone is unavailable, all VMs go down.
B. all virtual machines in a single Availability Set: Availability Sets protect against failures within a datacenter (e.g., hardware or rack failures) but do not help if the entire datacenter (Availability Zone) goes down.
D. each virtual machine in a separate Availability Set: Availability Sets cannot span multiple datacenters, so this does not provide resilience if a whole datacenter fails.
