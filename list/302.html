Question : 50 Topic: 4

You have an Azure virtual machine named VM1 that runs Windows Server 2019.

You save VM1 as a template named Template1 to the Azure Resource Manager library.
You plan to deploy a virtual machine named VM2 from Template1.
What can you configure during the deployment of VM2?

  • A..operating system
  • CorrectB..administrator username
  • C..virtual machine size
  • CorrectD..resource group
Suggested Answer:
B. administrator username
D. resource group
-----------------
Explanation of correct options:
administrator username: When deploying a VM from an Azure Resource Manager template, you must specify the administrator username as a required parameter during deployment.
resource group: During deployment, you are required to select or create a resource group for the new VM.
-----------------
Explanation of other options:
A. operating system: The operating system is defined by the source VM when you create the template; you cannot change the OS during deployment from that template.
C. virtual machine size: The VM size can be parameterized, but it is not mandatory to configure or change it during deployment; you can use the default size defined in the template if not specified.
Additional:
When deploy a VM from a template, you must specify:
- the Resource Group name and location for the VM
- the administrator username and password
- an unique DNS name for the public IP
