Question : 31 Topic: 1

Your company has two on-premises servers named SRV01 and SRV02. Developers have created an application that runs on SRV01. The application calls a service on SRV02 by IP address.

You plan to migrate the application on Azure virtual machines (VMs). You have configured two VMs on a single subnet in an Azure virtual network.
You need to configure the two VMs with static internal IP addresses.
What should you do?

  • A..Run the New-AzureRMVMConfig PowerShell cmdlet.
  • B..Run the Set-AzureSubnet PowerShell cmdlet.
  • C..Modify the VM properties in the Azure Management Portal.
  • D..Modify the IP properties in Windows Network and Sharing Center.
  • CorrectE..Run the Set-AzureStaticVNetIP PowerShell cmdlet.
Suggested Answer:
E.. Run the Set-AzureStaticVNetIP PowerShell cmdlet.
-----------------
Explanation of Correct Option:
E.. Run the Set-AzureStaticVNetIP PowerShell cmdlet. — This is the correct answer. To assign a static internal IP address to an Azure VM, you use the Set-AzureStaticVNetIP cmdlet. This ensures the VM retains the same private IP address across reboots and deallocations, which is essential for applications that rely on fixed IPs.
-----------------
Explanation of Other Options:
A.. Run the New-AzureRMVMConfig PowerShell cmdlet. — Incorrect. This cmdlet is used to create a new VM configuration object, not to assign static IP addresses.
B.. Run the Set-AzureSubnet PowerShell cmdlet. — Incorrect. This cmdlet is used to configure subnet settings, not individual VM IP addresses.
C.. Modify the VM properties in the Azure Management Portal. — Incorrect. While you can view IP settings in the portal, assigning a static private IP is typically done through PowerShell or the Azure CLI. Another select Networking instead of VM property in portal to setup the static ip address of a VM
D.. Modify the IP properties in Windows Network and Sharing Center. — Incorrect. Manually setting the IP inside the VM OS can cause conflicts with Azure's network management and is not supported for static IP assignment in Azure.
