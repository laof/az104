You have an Azure subscription named Subscription1 that is used by several departments at your company. Subscription1 contains the resources in the following table:

Another administrator deploys a virtual machine named VM1 and an Azure Storage account named storage2 by using a single Azure Resource Manager template.
You need to view the template used for the deployment.
From which blade can you view the template that was used for the deployment?

  • A..VM1
  • CorrectB..RG1
  • C..storage2
  • D..container1
Suggested Answer:
B..RG1
Explanation of right option:
RG1: The Resource Group blade contains the deployment history for resources created by an Azure Resource Manager template. You can view the template used for the deployment by navigating to the resource group (RG1), then selecting "Deployments" and choosing the specific deployment to see the template.
-----------------
Explanation of other options:
VM1: The Virtual Machine blade does not show the full deployment template; it only shows details related to the VM.
storage2: The Storage Account blade does not provide access to the ARM template used for deployment.
container1: The Blob Container blade is only for data management and does not show deployment templates.
