Your company has an Azure Active Directory (Azure AD) subscription.

You need to deploy five virtual machines (VMs) to your company's virtual network subnet.
The VMs will each have both a public and private IP address. Inbound and outbound security rules for all of these virtual machines must be identical.
Which of the following is the least amount of network interfaces needed for this configuration?

  • CorrectA..5
  • B..10
  • C..20
  • D..40
Suggested Answer:
A. 5
-----------------
Explanation of Correct Option:
A. 5 — This is the correct answer. Each Azure virtual machine (VM) only requires one network interface (NIC) to support both a private IP and a public IP. Azure allows assigning both types of IP addresses to a single NIC. Since there are five VMs, the least number of NICs required is 5 — one per VM.
-----------------
Explanation of Other Options:
B. 10 — Incorrect. This would imply two NICs per VM, which is unnecessary because a single NIC can handle both public and private IPs.
C. 20 — Incorrect. This would mean four NICs per VM, which is excessive and not required for this configuration.
D. 40 — Incorrect. This would mean eight NICs per VM, which is far beyond what is needed and not justified by the scenario.
