HOTSPOT -

You have an Azure subscription that contains the resources shown in the following table.
In Azure Cloud Shell, you need to create a virtual machine by using an Azure Resource Manager (ARM) template.
How should you complete the command? To answer, select the appropriate options in the answer area.
NOTE: Each correct selection is worth one point.
Hot Area:

Suggested Answer:
New-AzResourceGroupDeployment
-ResourceGroupName RG1
New-AzResourceGroupDeployment - This cmdlet is used to deploy Azure Resource Manager (ARM) templates to a resource group, which matches the requirement in the scenario to create a VM using an ARM template within a specific resource group.
-ResourceGroupName RG1 - The virtual machine must be deployed into a resource group, and "RG1" is the name of the resource group provided in the scenario.
-----------------
New-AzVm, New-AzResource, New-AzTemplateSpec - New-AzVm is used for creating VMs directly, not for deploying ARM templates. New-AzResource is for managing individual resources, not for template deployment. New-AzTemplateSpec is used to manage template specifications, not for deploying templates to resource groups.
-Tag Tag1, -GroupName ManagementGroup1, -Subscription 9c8bc1cd-7655-4c66-b3ea-a8ee101d8f75 - -Tag Tag1 adds a tag but does not specify deployment location. -GroupName ManagementGroup1 and -Subscription ... are not used to target deployments at the resource group level, which is what is required here.
