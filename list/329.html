Question : 77 Topic: 4

You develop the following Azure Resource Manager (ARM) template to create a resource group and deploy an Azure Storage account to the resource group.

Which cmdlet should you run to deploy the template?

  • A..New-AzResource
  • B..New-AzResourceGroupDeployment
  • C..New-AzTenantDeployment
  • CorrectD..New-AzDeployment
Suggested Answer:
D..New-AzDeployment
D..New-AzDeployment – This is the correct cmdlet to use because New-AzDeployment deploys ARM templates at the subscription scope, which allows the template to create the resource group (RG1) if it does not already exist, and then deploy resources into it. This matches the scenario shown, where RG1 does not exist before the deployment.
-----------------
A..New-AzResource
A..New-AzResource – Incorrect because this cmdlet is for creating or updating a single Azure resource, not for deploying ARM templates.
B..New-AzResourceGroupDeployment
B..New-AzResourceGroupDeployment – Incorrect because the resource group (RG1) must already exist before using this cmdlet. In the scenario, the template is responsible for creating RG1.
C..New-AzTenantDeployment
C..New-AzTenantDeployment – Incorrect because this cmdlet is used for deployments at the tenant scope, which is not needed for this scenario.
