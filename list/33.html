Question : 33 Topic: 1

Your company has an Azure Active Directory (Azure AD) subscription.

You need to deploy five virtual machines (VMs) to your company's virtual network subnet.
The VMs will each have both a public and private IP address. Inbound and outbound security rules for all of these virtual machines must be identical.
Which of the following is the least amount of security groups needed for this configuration?

  • A..4
  • B..3
  • C..2
  • CorrectD..1
Suggested Answer:
D. 1
-----------------
Explanation of Correct Option:
D. 1 — This is the correct answer. Since all five virtual machines require identical inbound and outbound security rules, you can associate a single Network Security Group (NSG) with either the subnet or each VM’s network interface. Using one NSG simplifies management and ensures consistent security policies across all VMs.
-----------------
Explanation of Other Options:
A. 4 — Incorrect. This would mean using multiple NSGs unnecessarily, increasing complexity without any added benefit.
B. 3 — Incorrect. Still more than needed. One NSG is sufficient for identical rules.
C. 2 — Incorrect. Two NSGs might be used if there were different rule sets, but in this case, all VMs share the same rules, so only one is needed.
