Question : 78 Topic: 4

HOTSPOT -

You have an Azure App Service app named WebApp1 that contains two folders named Folder1 and Folder2.
You need to configure a daily backup of WebApp1. The solution must ensure that Folder2 is excluded from the backup.
What should you create first, and what should you use to exclude Folder2? To answer, select the appropriate options in the answer area.
NOTE: Each correct selection is worth one point.
Hot Area:

Suggested Answer:
First create: An Azure Storage account
To exclude Folder2, use: A _backup.filter file
Explanation of right options:
An Azure Storage account – To configure backups for an Azure App Service (WebApp1), you must first create an Azure Storage account to store the backup files. This is a required prerequisite for enabling backup operations in App Service.
A _backup.filter file – To exclude specific folders (such as Folder2) from the backup, you use a special file named _backup.filter placed in the app’s directory. This file allows you to specify files and folders that should not be included in the backup.
-----------------
A Backup vault – This is used for Azure Backup but not for App Service backups; it is not necessary for configuring Web App backups.
A Recovery Services vault – This is also for Azure Backup and not applicable to App Service backups.
A resource group – While a resource group is needed for logical organization, it is not specifically required as the first step for enabling backups.
A backup policy – Backup policies are used in Azure Backup, not for excluding folders in App Service backups.
A lock – Locks prevent deletion or modification of resources and are not used for backup exclusions.
A WebJob – WebJobs are used for running background tasks and are not used for backup configurations or exclusions.
Additional:
You need a Recovery service vault if you want to backup VMs, File Shares, SAP HANA in a VM or SQL Server in a VM.
You need a Backup vault if you want to backup Azure Disks, Azure Blobs or Azure Database for PostgreSQL Server.
The question asks about an App Service, this one backs up to a storage account.
