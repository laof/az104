HOTSPOT -

You have an Azure subscription. The subscription contains a virtual machine that runs Windows 10.
You need to join the virtual machine to an Active Directory domain.
How should you complete the Azure Resource Manager (ARM) template? To answer, select the appropriate options in the answer area.
NOTE: Each correct selection is worth one point.
Hot Area:

Suggested Answer:
Type: Microsoft.Compute/virtualMachines/extensions
For the password property, use: ProtectedSettings
Explanation of right options:
Microsoft.Compute/virtualMachines/extensions – This is the correct resource type to use when deploying VM extensions, such as the JsonADDomainExtension, which is used to join a VM to an Active Directory domain.
ProtectedSettings – The password for joining the domain is sensitive information and must be placed inside the ProtectedSettings section. This ensures that the value is secured and not exposed in ARM template outputs or logs.
-----------------
Extensions – This is not a valid resource type in Azure Resource Manager templates.
Microsoft.Compute/VirtualMachines – This is the resource type for creating a virtual machine, but not for adding extensions to the VM.
Settings – The Settings property is used for non-sensitive configuration data, not for sensitive values like passwords.
Statuses – This property is not used for configuration or secret management in ARM templates.
