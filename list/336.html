HOTSPOT

-
You have an Azure subscription.
You create the following file named Deploy.json.
You connect to the subscription and run the following commands.
For each of the following statements, select Yes if the statement is true. Otherwise, select No.
NOTE: Each correct selection is worth one point.

Suggested Answer:
The commands will create four new resources. Yes
The commands will create storage accounts in the West US Azure region. No
The first storage account that is created will have a prefix of 0. Yes
Explanation of right options:
Yes (The commands will create four new resources.): The term "commands" includes both New-AzResourceGroup (which creates the resource group) and New-AzResourceGroupDeployment (which creates three storage accounts as defined in the ARM template). In total, four resources are created: one resource group and three storage accounts.
No (The commands will create storage accounts in the West US Azure region.): The resource group is created in "centralus" and the template uses [resourceGroup().location], so the storage accounts are created in Central US, not West US.
Yes (The first storage account that is created will have a prefix of 0.): The storage account name uses concat(copyIndex(),'storage',...), so the first resource will have a prefix of 0 (e.g., "0storage...").
-----------------
No (The commands will create four new resources.): Incorrect because the commands do create four resources in total, as explained.
Yes (The commands will create storage accounts in the West US Azure region.): Incorrect because the region is actually Central US, inherited from the resource group location.
No (The first storage account that is created will have a prefix of 0.): Incorrect because copyIndex() starts from 0, so the prefix for the first account will be 0.
