HOTSPOT

-
You have an Azure subscription that contains a resource group named RG1.
You plan to use an Azure Resource Manager (ARM) template named template1 to deploy resources. The solution must meet the following requirements:
• Deploy new resources to RG1.
• Remove all the existing resources from RG1 before deploying the new resources.
How should you complete the command? To answer, select the appropriate options in the answer area.
NOTE: Each correct selection is worth one point.

Suggested Answer:
1. ResourceGroupName
2. Complete
Explanation of right options:
ResourceGroupName: This parameter specifies the resource group (RG1) that the deployment should target. It matches the requirement to deploy to the resource group RG1.
Complete: The Complete deployment mode ensures that any resources in the resource group that are not specified in the template will be deleted. This meets the requirement to "remove all the existing resources from RG1 before deploying the new resources."
-----------------
Name, QueryString, Tag: These are incorrect because they do not specify the resource group for deployment.
All, Incremental: "All" is not a valid deployment mode. "Incremental" is the default and would only add/update resources, not remove existing resources not specified in the template.
Additional:
-Mode
Specifies the deployment mode. The acceptable values for this parameter are:
• Complete: In complete mode, Resource Manager deletes resources that exist in the resource group but are not specified in the template.
• Incremental: In incremental mode, Resource Manager leaves unchanged resources that exist in the resource group but are not specified in the template.
