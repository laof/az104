Question : 34 Topic: 1

Your company's Azure subscription includes Azure virtual machines (VMs) that run Windows Server 2016.

One of the VMs is backed up every day using Azure Backup Instant Restore.
When the VM becomes infected with data encrypting ransomware, you decide to recover the VM's files.
Which of the following is TRUE in this scenario?

  • A..You can only recover the files to the infected VM.
  • CorrectB..You can recover the files to any VM within the company's subscription.
  • C..You can only recover the files to a new VM.
  • D..You will not be able to recover the files.
Suggested Answer:
B..You can recover the files to any VM within the company's subscription.
-----------------
Explanation of Correct Option:
B..You can recover the files to any VM within the company's subscription. — This is the correct answer. Azure Backup Instant Restore allows you to mount the recovery point as a drive on any VM in the subscription. This enables you to browse and copy files from the backup without restoring the entire VM, making it ideal for recovering from ransomware attacks.
-----------------
Explanation of Other Options:
A..You can only recover the files to the infected VM. — Incorrect. Azure Backup supports file recovery to any VM, not just the original one.
C..You can only recover the files to a new VM. — Incorrect. File-level recovery does not require creating a new VM; it can be done on any existing VM.
D..You will not be able to recover the files. — Incorrect. Azure Backup provides recovery options specifically designed to handle scenarios like ransomware infections.
