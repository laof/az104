Question : 88 Topic: 4

You have an Azure subscription.

You plan to deploy the Azure container instances shown in the following table.
Which instances can you deploy to a container group?

  • A..Instance1 only
  • B..Instance2 only
  • C..Instance1 and Instance2 only
  • CorrectD..Instance3 and Instance4 only
Suggested Answer:
D. Instance3 and Instance4 only
-----------------
Explanation of Correct Option:
Azure Container Instances require that all containers in a container group use the same operating system and kernel type. Both Instance3 and Instance4 use Linux, so they can be grouped together.
Explanation of Other Options:
A. Instance1 only
Instance1 uses Nano Server installation of Windows Server 2019. While it is a valid Windows OS, the question asks which instances can be grouped, and grouping only one container is not meaningful for container groups.
B. Instance2 only
Instance2 uses Server Core installation of Windows Server 2019. Like option A, this is a single container and does not represent a group.
C. Instance1 and Instance2 only
Instance1 and Instance2 use different Windows base images (Nano Server and Server Core). Azure does not allow mixing these two base images in the same container group.
Additional:
Multi-container groups currently support only Linux containers.
For Windows containers, Azure Container Instances only supports deployment of a single container instance.
