HOTSPOT

-
You have an Azure subscription.
You need to deploy a virtual machine by using an Azure Resource Manager (ARM) template.
How should you complete the template? To answer, select the appropriate options in the answer area.
NOTE: Each correct selection is worth one point.

Suggested Answer:
resourceId
imageReference
-----------------
Explanation of Correct Options:
resourceId: In ARM templates, the dependsOn field should reference resources using the resourceId function. This ensures the correct dependency between the virtual machine and the network interface.
imageReference: The storageProfile section of a VM ARM template uses imageReference to specify the publisher, offer, SKU, and version of the image when deploying from a marketplace image.
Explanation of Other Options:
reference: The reference function is used to retrieve runtime properties of a resource, not for establishing dependencies.
Union: Union is not a valid function for dependencies in ARM templates.
Array: Array is not a valid property for storageProfile.
Image: Image is not the correct property name; the correct property is imageReference.
vhd: vhd is used when specifying a custom VHD, not when deploying from a marketplace image as in this template.
