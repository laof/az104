Question : 94 Topic: 4

You plan to create the Azure web apps shown in the following table.

What is the minimum number of App Service plans you should create for the web apps?

  • A..1
  • CorrectB..2
  • C..3
  • D..4
Suggested Answer:
B. 2

Explanation

You need a minimum of 2 App Service plans: one for Windows-based runtimes (.NET 6 (LTS), ASP.NET V4.8) and one for Linux-based runtimes (PHP 8.1, Python 3.11).
WebApp1 and WebApp2 can share a Windows plan;
WebApp3 and WebApp4 can share a Linux plan.
Mixing Windows and Linux web apps on the same plan is not supported.

Additional:

From Azure documentation:
ASP.NET Core (on Windows or Linux),
ASP.NET (on Windows),
PHP (on Windows or Linux),
Ruby (on Linux),
Node.js (on Windows or Linux),
Java (on Windows or Linux),
Python (on Linux),
HTML, Custom container (Windows or Linux)
