HOTSPOT -

You have an Azure subscription that contains the resource groups shown in the following table.
You create the following Azure Resource Manager (ARM) template named deploy.json.
You deploy the template by running the following cmdlet.
For each of the following statements, select Yes if the statement is true. Otherwise, select No.
NOTE: Each correct selection is worth one point.

Suggested Answer:
1. The template creates a resource group named RG0 in the East US Azure region. — Yes
(The ARM template loops and creates RG0–RG3 all in "eastus".)
2. The template creates four new resource groups. — No
(RG1 already exists in East US, so only RG0, RG2, and RG3 are newly created; RG1 is reused and not new.)
3. The template creates a resource group named RG3 in the West US Azure region. — No
(All resource groups are created in "eastus" as defined in the template, not "westus".)
