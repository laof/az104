You have an Azure App Service app named App1 that contains two running instances.

You have an autoscale rule configured as shown in the following exhibit.
For the Instance limits scale condition setting, you set Maximum to 5.
During a 30-minute period, App1 uses 80 percent of the available memory.
What is the maximum number of instances for App1 during the 30-minute period?

  • A..2
  • B..3
  • C..4
  • CorrectD..5
Suggested Answer:Answer: D. 5
(The autoscale rule increases the instance count by 1 if the average memory percentage is greater than 70% for 15 minutes, with a cool down of 5 minutes. Since the scale maximum is set to 5, starting from 2 instances and with sufficient memory usage, the app can scale up to the maximum of 5 instances during the 30-minute period.)
