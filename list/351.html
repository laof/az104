Question : 99 Topic: 4

You have an Azure subscription.

You plan to deploy the resources shown in the following table.
You need to create a single Azure Resource Manager (ARM) template that will be used to deploy the resources.
Which resource should be added to the dependsOn section for VM1?

  • A..VNET1
  • CorrectB..NIC1
  • C..IP1
  • D..NSG1
Suggested Answer:
B..NIC1 (A virtual machine (VM1) in Azure must reference a network interface (NIC1) at deployment, so the VM must depend on the network interface being created first. Therefore, NIC1 should be added to the dependsOn section for VM1 in the ARM template.)
-----------------
A..VNET1 (Although the network interface (NIC1) is associated with the virtual network (VNET1), VM1 does not directly depend on the virtual network; it depends on the network interface, which in turn depends on the VNET.)
C..IP1 (The public IP address, if needed, is associated with the network interface, not directly with the VM. Therefore, the VM does not need to directly depend on IP1.)
D..NSG1 (The network security group (NSG1) is also associated with the network interface or subnet, not directly with the VM. VM1 does not require a direct dependency on NSG1.)
