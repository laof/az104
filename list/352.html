You have an Azure subscription.

You create the following Azure Resource Manager (ARM) template named Template.json.
You need to deploy Template.json.
Which PowerShell cmdlet should you run from Azure Cloud Shell?

  • CorrectA..New-AzSubscriptionDeployment
  • B..New-AzManagementGroupDeployment
  • C..New-AzResourceGroupDeployment
  • D..New-AzTenantDeployment
Suggested Answer:
A..New-AzSubscriptionDeployment (The template defines a resource group as a resource, which requires a deployment at the subscription scope. The correct cmdlet for deploying ARM templates that create resource groups is New-AzSubscriptionDeployment.)
-----------------
B..New-AzManagementGroupDeployment (This cmdlet is used for deployments targeting management groups, not for creating resource groups at the subscription level.)
C..New-AzResourceGroupDeployment (This cmdlet deploys resources into an existing resource group, but your template creates a new resource group, so this is not appropriate.)
D..New-AzTenantDeployment (This cmdlet is for tenant-level deployments, which is not required for this template that targets subscription-level resource group creation.)
