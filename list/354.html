Question : 102 Topic: 4

HOTSPOT

-
Your company purchases a new Azure subscription.
You create a file named Deploy.json as shown in the following exhibit.
You connect to the subscription and run the following cmdlet.
New-AzDeployment -Location westus -TemplateFile “deploy.json”
For each of the following statements, select Yes if the statement is true. Otherwise, select No.
NOTE: Each correct selection is worth one point.

Suggested Answer:
A..You can deploy a virtual machine to RG1. Yes
B..You can deploy a virtual machine to RG2. No
C..You can manually create a resource group named RG3. Yes
-----------------
Explanation of right options:
A..You can deploy a virtual machine to RG1. Yes — The CanNotDelete lock on RG1 only prevents deletion of resources within the resource group; it does not prevent creating or updating resources. Therefore, you can deploy a virtual machine to RG1.
B..You can deploy a virtual machine to RG2. No — The ReadOnly lock on RG2 prevents creating, updating, or deleting resources within RG2. Therefore, you cannot deploy a VM to RG2.
C..You can manually create a resource group named RG3. Yes — Locks applied to RG1 and RG2 do not restrict actions outside those groups. You can manually create a new resource group like RG3.
