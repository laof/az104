HOTSPOT

-
You have an Azure subscription that contains the virtual networks shown in the following table.
The subscription contains the virtual machines shown in the following table.
The subscription contains the Azure App Service web apps shown in the following table.
For each of the following statements, select Yes if the statement is true. Otherwise, select No.
NOTE: Each correct selection is worth one point.

Suggested Answer:
A..WebApp1 can communicate with VM2. Yes
B..NSG1 controls inbound traffic to WebApp1. No
C..WebApp2 can communicate with VM1. Yes
-----------------
Explanation of right options:
A..WebApp1 can communicate with VM2. Yes — WebApp1 uses the Premium pricing tier and is integrated with VNet1. VNet1 is peered with VNet2, and VM2 is in Subnet2 of VNet2. Premium VNet Integration allows outbound connectivity from WebApp1 to resources in both the integrated VNet and peered VNets, so WebApp1 can communicate with VM2.
B..NSG1 controls inbound traffic to WebApp1. No — NSGs control network traffic to subnets and VMs, but they do not control traffic to Azure App Services like WebApp1. Inbound HTTP/HTTPS traffic to WebApp1 is not affected by NSG1.
C..WebApp2 can communicate with VM1. Yes — WebApp2 is deployed to Subnet2 in VNet2. VM1 is in Subnet1 in VNet1. The two VNets are peered, so communication between resources across both VNets is possible.
