You have an Azure subscription that contains an Azure container registry named ContReg1.

You enable the Admin user for ContReg1.
Which username can you use to sign in to ContReg1?

  • A..root
  • B..admin
  • C..administrator
  • CorrectD..ContReg1
Suggested Answer:
D..ContReg1
-----------------
Explanation of right option:
D..ContReg1 — When you enable the Admin user for an Azure Container Registry, the Username shown in the portal is the registry name itself (in this case, ContReg1), as evidenced by the Access keys screenshot. You must use this value as the admin username when signing in.
Other options explained:
A..root No — "root" is not a valid admin username for Azure Container Registry.
B..admin No — Although some documentation references "admin" as a generic admin username, the actual username is shown in the portal and is typically the registry name, not "admin".
C..administrator No — "administrator" is not a valid username for Azure Container Registry admin login.
