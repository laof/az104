Question : 36 Topic: 1

You administer a solution in Azure that is currently having performance issues.

You need to find the cause of the performance issues pertaining to metrics on the Azure infrastructure.
Which of the following is the tool you should use?

  • A..Azure Traffic Analytics
  • CorrectB..Azure Monitor
  • C..Azure Activity Log
  • D..Azure Advisor
Suggested Answer:
B. Azure Monitor
-----------------
Explanation of Correct Option:
B. Azure Monitor — This is the correct answer. Azure Monitor is the primary tool for collecting, analyzing, and acting on telemetry data from Azure resources. It provides detailed performance metrics and diagnostics that help identify and resolve infrastructure-related issues.
-----------------
Explanation of Other Options:
A. Azure Traffic Analytics — Incorrect. This tool is focused on analyzing network traffic patterns and security, not general performance metrics.
C. Azure Activity Log — Incorrect. This log tracks control-plane operations (like resource creation or deletion), not performance metrics.
D. Azure Advisor — Incorrect. Azure Advisor provides recommendations for best practices, cost optimization, and security, but it does not offer real-time performance metrics.
