You have an Azure subscription that has the Azure container registries shown in the following table.
You plan to use ACR Tasks and configure private endpoint connections.
Which container registries support ACR Tasks and private endpoints? To answer, select the appropriate options in the answer area.
NOTE: Each correct selection is worth one point.

Suggested Answer:
ACR Tasks: ContReg1, ContReg2, and ContReg3
Private endpoints: ContReg1 and ContReg2 only
-----------------
Explanation of right options:
ACR Tasks: ContReg1, ContReg2, and ContReg3 — ACR Tasks are supported on all service tiers (Premium, Standard, and Basic), so all listed registries (ContReg1, ContReg2, ContReg3) support ACR Tasks.
Private endpoints: ContReg1 and ContReg2 only — Private endpoints are supported only for Premium and Standard tiers. The Basic tier (ContReg3) does not support private endpoints.
