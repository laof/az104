Question : 111 Topic: 4

You have an Azure subscription.

You plan to deploy a container.
You need to recommend which Azure services can scale the container automatically.
What should you recommend?

  • A..Azure Container Apps only
  • B..Azure Container Instances only
  • CorrectC..Azure Container Apps or Azure App Service only
  • D..Azure Container Instances or Azure App Service only
  • E..Azure Container Apps, Azure Container Instances, or Azure App Service
Suggested Answer:
C..Azure Container Apps or Azure App Service only
-----------------
Explanation of right option:
C..Azure Container Apps or Azure App Service only — Azure Container Apps and Azure App Service both support automatic scaling for containers. Azure Container Apps can scale based on HTTP traffic, event-driven triggers, or CPU/memory usage. Azure App Service also provides built-in autoscaling for container-based web apps.
Other options explained:
A..Azure Container Apps only — While Azure Container Apps supports autoscaling, App Service does too, so this is incomplete.
B..Azure Container Instances only — Azure Container Instances do not support autoscaling; they are for single-instance, on-demand containers.
D..Azure Container Instances or Azure App Service only — Azure Container Instances do not support autoscaling.
E..Azure Container Apps, Azure Container Instances, or Azure App Service — Azure Container Instances do not support autoscaling, so this list is incorrect.
