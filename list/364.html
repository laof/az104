Question : 112 Topic: 4

HOTSPOT

-
You have an Azure subscription that uses Azure Container Instances.
You have a computer that has Azure Command-Line Interface (CLI) and Docker installed.
You create a container image named image1.
You need to provision a new Azure container registry and add image1 to the registry.
Which command should you run for each requirement? To answer, select the options in the answer area.
NOTE: Each correct answer is worth one point.

Suggested Answer:
Provision a new container registry: az acr create
Add image1 to the registry: docker push
-----------------
Explanation of right options:
az acr create — The az acr create command provisions (creates) a new Azure Container Registry using the Azure CLI.
docker push — The docker push command uploads (pushes) a local Docker image (like image1) to a container registry.
Other options explained:
az acr build — This command is used to build container images in Azure, not for creating registries.
az container create — This command creates a container instance, not a registry or uploads images.
docker create — This command creates a container from an image, not a registry.
az acr create (for "add image1 to the registry") — This creates a registry, not for adding images. docker pull — This command is for downloading images from a registry, not uploading.
