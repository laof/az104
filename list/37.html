Question : 37 Topic: 1

Your company has an Azure subscription that includes a Recovery Services vault.

You want to use Azure Backup to schedule a backup of your company's virtual machines (VMs) to the Recovery Services vault.
Which of the following VMs can you back up? Choose all that apply.

  • CorrectA..VMs that run Windows 10.
  • CorrectB..VMs that run Windows Server 2012 or higher.
  • CorrectC..VMs that have NOT been shut down.
  • CorrectD..VMs that run Debian 8.2+.
  • CorrectE..VMs that have been shut down.
Suggested Answer:
A. VMs that run Windows 10.
B. VMs that run Windows Server 2012 or higher.
C. VMs that have NOT been shut down.
D. VMs that run Debian 8.2+.
E. VMs that have been shut down.
-----------------
Explanation of Correct Options:
A. VMs that run Windows 10. — Supported. Azure Backup supports Windows 10 (64-bit) for backup using the VM agent extension.
B. VMs that run Windows Server 2012 or higher. — Supported. These are fully supported for Azure Backup.
C. VMs that have NOT been shut down. — Supported. Azure Backup supports running VMs and can take application-consistent snapshots.
D. VMs that run Debian 8.2+. — Supported. Debian 8.2 and newer are among the endorsed Linux distributions for Azure VM backup.
E. VMs that have been shut down. — Supported. Azure Backup can back up shut down or offline VMs, though the snapshot will be crash-consistent only.
