You have an Azure virtual machine named VM1. VM1 was deployed by using a custom Azure Resource Manager template named ARM1.json.
You receive a notification that VM1 will be affected by maintenance.
You need to move VM1 to a different host immediately.
Solution: From the VM1 Redeploy + reapply blade, you select Redeploy.
Does this meet the goal?

  • CorrectA..Yes
  • B..No
Suggested Answer:
A..Yes
-----------------
Explanation of right option:
A..Yes — Using the Redeploy feature in Azure will move the VM to a new physical host within the same region, which immediately addresses the requirement to move VM1 to a different host when notified of maintenance.
-----------------
Other options explained:
B..No — Incorrect. Selecting Redeploy from the VM1 blade does in fact move the VM to a new host and meets the goal.
