HOTSPOT -

You plan to deploy five virtual machines to a virtual network subnet.
Each virtual machine will have a public IP address and a private IP address.
Each virtual machine requires the same inbound and outbound security rules.
What is the minimum number of network interfaces and network security groups that you require? To answer, select the appropriate options in the answer area.
NOTE: Each correct selection is worth one point.
Hot Area:

Suggested Answer:
Minimum number of network interfaces: 5
— Each virtual machine must have at least one network interface, so for five VMs, the minimum required is five network interfaces.
Minimum number of network security groups: 1
— Since all virtual machines require the same inbound and outbound security rules, you can associate a single network security group (NSG) with all five network interfaces or the subnet. Only one NSG is needed.
-----------------
Other options explained:
10, 15, 20 network interfaces — Incorrect. Each VM only needs one network interface. More than five would be unnecessary unless each VM needed multiple NICs, which is not required in this scenario.
2, 5, 10 network security groups — Incorrect. Multiple NSGs are not required since the same rules apply to all VMs; using more increases complexity but doesn't provide additional value here.
