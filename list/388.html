Question : 15 Topic: 5

You create an Azure VM named VM1 that runs Windows Server 2019.

VM1 is configured as shown in the exhibit. (Click the Exhibit tab.)
You need to enable Desired State Configuration for VM1.
What should you do first?

  • A..Connect to VM1.
  • CorrectB..Start VM1.
  • C..Capture a snapshot of VM1.
  • D..Configure a DNS name for VM1.
Suggested Answer:
B. Start VM1.
This is the correct option. The VM status is currently Stopped (deallocated), as shown in the exhibit. Before you can enable Desired State Configuration (DSC) or perform any management actions, the VM must be running. Starting the VM will allocate resources and allow you to configure and manage it, including enabling DSC.
-----------------
A. Connect to VM1.
You cannot connect to the VM while it is stopped/deallocated. The VM must be running to establish a connection.
C. Capture a snapshot of VM1.
Capturing a snapshot is unrelated to enabling Desired State Configuration. Snapshots are used for backup and recovery, not configuration management.
D. Configure a DNS name for VM1.
Configuring a DNS name is not required to enable Desired State Configuration. The VM must be running before any configuration can be applied.
Additional:
The DSC extension for Windows requires that the target virtual machine is able to communicate with Azure.
