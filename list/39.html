You have an Azure Active Directory (Azure AD) tenant named contoso.com.
You have a CSV file that contains the names and email addresses of 500 external users.
You need to create a guest user account in contoso.com for each of the 500 external users.
Solution: From Azure AD in the Azure portal, you use the Bulk create user operation.
Does this meet the goal?

  • A..Yes
  • CorrectB..No
Suggested Answer:
B. No
-----------------
Explanation of Correct Option:
B. No — This is the correct answer. The Bulk create user operation in Azure AD is used to create internal users (members), not external guest users. To add external users from a CSV file, you must use the Bulk invite users operation, which sends invitations and creates guest accounts in the directory.
-----------------
Explanation of Other Option:
A. Yes — Incorrect. While the Bulk create user operation supports creating multiple users, it does not support the guest invitation process required for external users. The correct method is Bulk invite users, which is specifically designed for onboarding external (B2B) users.
