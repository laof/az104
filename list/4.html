Your company has an Azure Active Directory (Azure AD) subscription.
You want to implement an Azure AD conditional access policy.
The policy must be configured to require members of the Global Administrators group to use Multi-Factor Authentication and an Azure AD-joined device when they connect to Azure AD from untrusted locations.
Solution: You access the Azure portal to alter the grant control of the Azure AD conditional access policy.
Does the solution meet the goal?

  • CorrectA..Yes
  • B..No
Suggested Answer:
Correct Option: A. Yes
Explanation: Altering the grant control of the Azure AD conditional access policy meets the goal because it allows you to specify the required conditions for Multi-Factor Authentication and an Azure AD-joined device for Global Administrators connecting from untrusted locations. This configuration ensures that the policy is enforced correctly.
-----------------
Explanation of Other Options:
B. No: This option is incorrect because altering the grant control of the Azure AD conditional access policy is the correct method to configure the required conditions for the policy.

Additional:

Process to Configure an Azure AD Conditional Access Policy:
1. Access Azure AD Portal: Log in to the Azure AD portal with administrative credentials.
2. Navigate to Conditional Access: Go to the Security settings and select Conditional Access.
3. Create a New Policy: Click on "New policy" to start creating a policy.
4. Define Conditions: Specify the conditions under which the policy applies, such as user groups, applications, and locations.
5. Set Access Controls: Determine the access controls, like requiring Multi-Factor Authentication (MFA) and device compliance.
6. Enable and Save: Enable the policy and click "Create" to save it.
-----------------
Grant Control: The grant control in Azure AD conditional access policies allows you to specify requirements that must be met for access to be granted. This includes conditions like requiring Multi-Factor Authentication (MFA), compliant devices, or specific authentication methods. When configuring grant control, you can select multiple requirements that users must satisfy to access resources.
-----------------
Explanation: Grant control is essential for enforcing security measures and ensuring that only authenticated and compliant users can access sensitive resources. By setting up grant control, you can enhance the security posture of your organization and protect against unauthorized access.
