You plan to create an Azure virtual machine named VM1 that will be configured as shown in the following exhibit.

The planned disk configurations for VM1 are shown in the following exhibit.
You need to ensure that VM1 can be created in an Availability Zone.
Which two settings should you modify? Each correct answer presents part of the solution.
NOTE: Each correct selection is worth one point.

  • CorrectA..Use managed disks
  • B..OS disk type
  • CorrectC..Availability options
  • D..Size
  • E..Image
Suggested Answer:
A.. Use managed disks: Availability Zones require managed disks. You must change the setting to "Yes" for managed disks to deploy the VM in an Availability Zone.
C.. Availability options: You must select an "Availability Zone" under availability options to deploy the VM in an Availability Zone.
-----------------
B.. OS disk type: This affects the disk performance but does not enable deployment in an Availability Zone.
D.. Size: While only certain VM sizes are supported in Availability Zones, the main requirement is managed disks and the zone selection itself.
E.. Image: The OS image does not impact the ability to deploy in an Availability Zone; it is not a requirement for zone support.
