HOTSPOT -

You have an Azure subscription that contains the resources shown in the following table.
VMSS1 is set to VM (virtual machines) orchestration mode.
You need to deploy a new Azure virtual machine named VM1, and then add VM1 to VMSS1.
Which resource group and location should you use to deploy VM1? To answer, select the appropriate options in the answer area.
NOTE: Each correct selection is worth one point.
Hot Area:

Suggested Answer:
1. Resource group: RG1 only:
A VM added to a virtual machine scale set (VMSS1) must be in the same resource group as the scale set, which is RG1.
2. Location: West US only:
A VM must be in the same region as the scale set to be added to it. VMSS1 is in West US, so VM1 must also be in West US.
-----------------
Resource group: RG2 only, RG1 or RG2 only, RG1, RG2, or RG3: These options would allow the VM to be deployed in a different resource group than the scale set, which is not supported for adding the VM to VMSS1.
Location: Central US only, Central US or West US only, East US, Central US, or West US: These options would allow the VM to be in a different region than the scale set, which is not supported for adding the VM to VMSS1.
