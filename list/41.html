HOTSPOT -

You have an Azure subscription named Subscription1 that contains a resource group named RG1.
In RG1, you create an internal load balancer named LB1 and a public load balancer named LB2.
You need to ensure that an administrator named Admin1 can manage LB1 and LB2. The solution must follow the principle of least privilege.
Which role should you assign to Admin1 for each task? To answer, select the appropriate options in the answer area.
NOTE: Each correct selection is worth one point.
Hot Area:

Suggested Answer:
To add the backend pool to the load balancer resource, the user needs to have permissions to be able to read the virtual network and virtual machine resources that need to be associated to the backend pool. Hence permissions need to be given at the resource group level.
To read the virtual network and virtual machine resources, you need to have at least Network Contributor role. You need the same role to add a health probe.
