You have a computer named Computer1 that has a point-to-site VPN connection to an Azure virtual network named VNet1. The point-to-site connection uses a self-signed certificate.
From Azure, you download and install the VPN client configuration package on a computer named Computer2.
You need to ensure that you can establish a point-to-site VPN connection to VNet1 from Computer2.
Solution: You export the client certificate from Computer1 and install the certificate on Computer2.
Does this meet the goal?

  • CorrectA..Yes
  • B..No
Suggested Answer:
A..Yes — Exporting the client certificate from Computer1 and installing it on Computer2 allows Computer2 to authenticate and successfully establish a point-to-site VPN connection to VNet1 using the same certificate-based authentication.
-----------------
B..No — Incorrect: Installing the exported client certificate on Computer2 enables it to use the same authentication as Computer1, so this solution does meet the goal.
