You manage a virtual network named VNet1 that is hosted in the West US Azure region.
VNet1 hosts two virtual machines named VM1 and VM2 that run Windows Server.
You need to inspect all the network traffic from VM1 to VM2 for a period of three hours.
Solution: From Performance Monitor, you create a Data Collector Set (DCS).
Does this meet the goal?

  • A..Yes
  • CorrectB..No
Suggested Answer:
B..No
Explanation: Performance Monitor with a Data Collector Set (DCS) can collect system performance metrics (such as CPU, memory, disk, and network interface statistics), but it does not capture or inspect the actual network traffic (packets) between VM1 and VM2. To inspect all network traffic, you need a packet capture tool such as Azure Network Watcher's Packet Capture feature.
