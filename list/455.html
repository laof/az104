  • B..comp1.contoso.com, comp2.contoso.com, comp3.contoso.com, and comp4.contoso.com
  • CorrectC..comp2.contoso.com only

Suggested Answer:
C..comp2.contoso.com only â€” comp2.contoso.com is an A record pointing to 10.0.0.5 (VM2), allowing ping. Only A or CNAME-to-A records that resolve to VM2's IP can be used for ping.
comp1.contoso.com is a TXT record and comp3.contoso.com is a CNAME to a TXT, not an A record, so these cannot be used to ping VM2. comp4.contoso.com is a PTR record, which only maps IP to name (reverse lookup) and cannot be used to resolve the IP from the name for ping.

Explanation of other options:
B..comp1.contoso.com, comp2.contoso.com, comp3.contoso.com, and comp4.contoso.com is incorrect because comp1.contoso.com is a TXT record (not used for ping), comp3.contoso.com is a CNAME pointing to a TXT record (not an A record), and comp4.contoso.com is a PTR record for reverse lookups only.
D..comp1.contoso.com and comp2.contoso.com only is incorrect because comp1.contoso.com is not an A/CNAME-to-A record.
E..comp1.contoso.com, comp2.contoso.com, and comp4.contoso.com only is incorrect for the same reason; comp1.contoso.com and comp4.contoso.com cannot be used to ping VM2.
