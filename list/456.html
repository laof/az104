How should you configure the rule? To answer, select the appropriate options in the answer area.
NOTE: Each correct selection is worth one point.
Hot Area:

Suggested Answer:
Direction: Outbound
Source: 10.1.0.10
Destination: 10.1.0.11
Priority: 110
Explanation:
Since the DENY_PING rule is an outbound rule, you should add a new outbound allow rule with a higher priority (lower number, 110) to permit only ICMP traffic from VM1 (10.1.0.10) to VM2 (10.1.0.11), following the principle of least privilege; this allows ping from VM1 to VM2 while keeping all other ICMP traffic blocked as per the original deny rule.
