Question : 86 Topic: 5

You have an Azure subscription that uses the public IP addresses shown in the following table.

You need to create a public Azure Standard Load Balancer.
Which public IP addresses can you use?

  • A..IP1, IP2, and IP3
  • B..IP2 only
  • CorrectC..IP3 only
  • D..IP1 and IP3 only
Suggested Answer:
C..IP3 only
Explanation: Only public IP addresses with the Standard SKU can be used with an Azure Standard Load Balancer. IP3 is the only address with a Standard SKU, so only IP3 is valid.
Other Options Explained:
A. IP1, IP2, and IP3: Incorrect. IP1 and IP2 use the Basic SKU, which cannot be associated with a Standard Load Balancer.
B. IP2 only: Incorrect. IP2 uses the Basic SKU and is Dynamic; both are unsupported for Standard Load Balancers.
D. IP1 and IP3 only: Incorrect. IP1 is Basic SKU; only IP3 (Standard SKU) is supported.
