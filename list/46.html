Question : 6 Topic: 2

You have an Azure policy as shown in the following exhibit:

What is the effect of the policy?

  • A..You are prevented from creating Azure SQL servers anywhere in Subscription 1.
  • CorrectB..You can create Azure SQL servers in ContosoRG1 only.
  • C..You are prevented from creating Azure SQL Servers in ContosoRG1 only.
  • D..You can create Azure SQL servers in any resource group within Subscription 1.
Suggested Answer:
Question 2-6 Answer: B. You can create Azure SQL servers in ContosoRG1 only.
-----------------
Explanation of Correct Options: B. You can create Azure SQL servers in ContosoRG1 only. — The policy is scoped to Subscription 1 and blocks the creation of Microsoft.Sql/servers. However, ContosoRG1 is listed as an exclusion, meaning the policy does not apply there. Therefore, Azure SQL servers can only be created in ContosoRG1 and are blocked elsewhere in the subscription.
-----------------
Explanation of Other Options:
A. You are prevented from creating Azure SQL servers anywhere in Subscription 1. — Incorrect because ContosoRG1 is excluded from the policy, so creation is allowed there.
C. You are prevented from creating Azure SQL Servers in ContosoRG1 only. — Incorrect because ContosoRG1 is excluded, so the policy does not block creation there.
D. You can create Azure SQL servers in any resource group within Subscription 1. — Incorrect because the policy blocks creation in all resource groups except ContosoRG1.
