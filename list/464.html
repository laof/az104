Question : 91 Topic: 5

You have an Azure subscription that contains two virtual machines named VM1 and VM2.

You create an Azure load balancer.
You plan to create a load balancing rule that will load balance HTTPS traffic between VM1 and VM2.
Which two additional load balancer resources should you create before you can create the load balancing rule? Each correct answer presents part of the solution.
NOTE: Each correct selection is worth one point.

  • A..a frontend IP address
  • B..an inbound NAT rule
  • C..a virtual network
  • CorrectD..a backend pool
  • CorrectE..a health probe
Suggested Answer:
D..a backend pool
E..a health probe
Explanation: Since the load balancer has already been created (and thus has a frontend IP address), you only need to create a backend pool (to specify the VMs for traffic distribution) and a health probe (to monitor VM health) before adding a load balancing rule.
Other options:
A..a frontend IP address – This is created with the load balancer and already exists.
B..an inbound NAT rule – NAT rules are used for direct VM access, not for load balancing.
C..a virtual network – The VMs already exist, so the virtual network is already present.
