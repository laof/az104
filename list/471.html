Question : 98 Topic: 5

Your on-premises network contains an SMB share named Share1.

You have an Azure subscription that contains the following resources:
✑ A web app named webapp1
✑ A virtual network named VNET1
You need to ensure that webapp1 can connect to Share1.
What should you deploy?

  • A..an Azure Application Gateway
  • B..an Azure Active Directory (Azure AD) Application Proxy
  • CorrectC..an Azure Virtual Network Gateway
Suggested Answer:
C..an Azure Virtual Network Gateway
Explanation: To allow webapp1 to connect to the on-premises SMB share (Share1), you need to connect your Azure virtual network to your on-premises network using an Azure Virtual Network Gateway, enabling secure private connectivity between Azure and on-premises resources.
---------------------
A..an Azure Application Gateway
Explanation: Azure Application Gateway is for web traffic load balancing and HTTP/HTTPS routing, not for hybrid network connectivity or SMB shares.
B..an Azure Active Directory (Azure AD) Application Proxy
Explanation: Azure AD Application Proxy provides secure remote access to on-premises web applications, not SMB shares or direct network connectivity between Azure and on-premises networks.
