You have an Azure subscription that contains the resources shown in the following table.

You create a public IP address named IP1.
Which two resources can you associate to IP1? Each correct answer presents a complete solution.
NOTE: Each correct selection is worth one point.

  • A..VM1
  • CorrectB..LB1
  • CorrectC..NIC1
  • D..VPN1
  • E..VNet1
Suggested Answer:
B..LB1
C..NIC1
Explanation:
In Azure, a public IP address can be associated with a network interface (NIC1) or with a load balancer (LB1). You cannot directly assign a public IP to a virtual machine (VM1), virtual network (VNet1), or virtual network gateway (VPN1).
