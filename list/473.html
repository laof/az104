You have an Azure subscription that contains a storage account named storage1.

You need to allow access to storage1 from selected networks and your home office. The solution must minimize administrative effort.
What should you do first for storage1?

  • A..Add a private endpoint.
  • CorrectB..Modify the Public network access settings.
  • C..Select Internet routing.
  • D..Modify the Access Control (IAM) settings.
Suggested Answer:
B..Modify the Public network access settings
Explanation: To allow access to storage1 from selected networks and your home office with minimal effort, first modify the Public network access settings to specify allowed networks and IPs.
---------------------
A..Add a private endpoint
Explanation: A private endpoint restricts access to a virtual network and does not easily allow access from a public IP like your home office.
C..Select Internet routing
Explanation: Internet routing is used for private endpoint routing and does not control storage account network access.
D..Modify the Access Control (IAM) settings
Explanation: IAM controls management permissions, not network access to the storage account.
