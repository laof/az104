Question : 108 Topic: 5

You have five Azure virtual machines that run Windows Server 2016. The virtual machines are configured as web servers.

You have an Azure load balancer named LB1 that provides load balancing services for the virtual machines.
You need to ensure that visitors are serviced by the same web server for each request.
What should you configure?

  • A..Session persistence to None
  • B..a health probe
  • CorrectC..Session persistence to Client IP
  • D..Idle Time-out (minutes) to 20
Suggested Answer:
C..Session persistence to Client IP
Explanation: Setting Session persistence to Client IP ensures that requests from the same client IP are always directed to the same backend VM, maintaining session affinity for every visitor.
-----------------
A..Session persistence to None
Explanation: With no session persistence, requests from the same client can be handled by different backend VMs.
B..a health probe
Explanation: Health probes are used to determine the health of backend instances, not to control session affinity.
D..Idle Time-out (minutes) to 20
Explanation: Idle time-out affects connection duration, not session persistence or affinity.
