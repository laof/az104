You have an Azure subscription that contains the resources shown in the following table.

You need to manage outbound traffic from VNET1 by using Firewall1.
What should you do first?

  • A..Configure the Hybrid Connection Manager.
  • CorrectB..Upgrade ASP1 to the Premium SKU.
  • C..Create a route table.
  • D..Create an Azure Network Watcher.
Suggested Answer:
B..Upgrade ASP1 to the Premium SKU.
-----------------
A..Configure the Hybrid Connection Manager: Incorrect — Hybrid Connection Manager is used for hybrid connectivity scenarios but is not needed for managing outbound traffic from an app service integrated with a VNet.
B..Upgrade ASP1 to the Premium SKU: Correct — Outbound traffic routing (like using a route table with Azure Firewall) for Azure App Service requires the App Service plan to be Premium V2 or higher. The Standard SKU does not support forced tunneling or route table association for outbound traffic from the app service.
C..Create a route table: Incorrect — Creating a route table to direct traffic through Azure Firewall is necessary, but the App Service plan must be upgraded to Premium SKU first, as Standard SKU does not support this functionality.
D..Create an Azure Network Watcher: Incorrect — Network Watcher is used for monitoring and troubleshooting network issues; it does not enable or affect routing outbound traffic through the firewall.
