Question : 115 Topic: 5

You have an Azure subscription that contains the resources shown in the following table.

All the resources connect to a virtual network named VNet1.
You plan to deploy an Azure Bastion host named Bastion1 to VNet1.
Which resources can be protected by using Bastion1?

  • CorrectA..VM1 only
  • B..contoso.com only
  • C..App1 and contoso.com only
  • D..VM1 and contoso.com only
  • E..VM1, App1, and contoso.com
Suggested Answer:
A..VM1 only
-----------------
A..VM1 only: Correct — Azure Bastion is designed to provide secure RDP and SSH connectivity to virtual machines in an Azure virtual network, without exposing public IP addresses. Bastion cannot protect or provide access to web apps or Azure AD DS domains.
B..contoso.com only: Incorrect — Azure Bastion does not provide protection or direct connectivity for Azure AD DS (Domain Services) resources.
C..App1 and contoso.com only: Incorrect — Bastion cannot be used for web apps or Azure AD DS resources.
D..VM1 and contoso.com only: Incorrect — Only VM1 (the virtual machine) can be protected by Bastion; Azure AD DS cannot.
E..VM1, App1, and contoso.com: Incorrect — Bastion is not used for web apps or Azure AD DS, only for virtual machines.
