Question : 117 Topic: 5

You have five Azure virtual machines that run Windows Server 2016. The virtual machines are configured as web servers.

You have an Azure load balancer named LB1 that provides load balancing services for the virtual machines.
You need to ensure that visitors are serviced by the same web server for each request.
What should you configure?

  • A..a health probe
  • B..Floating IP (direct server return) to Enabled
  • CorrectC..Session persistence to Client IP and protocol
  • D..Protocol to UDP
Suggested Answer:
C..Session persistence to Client IP and protocol
-----------------
A..a health probe: Incorrect — A health probe determines which backend virtual machines are available and healthy, but does not ensure that the same web server handles each request from a client.
B..Floating IP (direct server return) to Enabled: Incorrect — Floating IP is used for scenarios such as SQL Always On or NLB clusters, but it does not control session persistence for client requests.
C..Session persistence to Client IP and protocol: Correct — Session persistence set to Client IP and protocol ensures that requests from the same client IP and protocol are always routed to the same backend VM, meeting the requirement for sticky sessions.
D..Protocol to UDP: Incorrect — Changing the protocol to UDP is not related to ensuring that clients are serviced by the same server for each request; session persistence configuration is required for that purpose.
