You have two Azure subscriptions named Sub1 and Sub2.

Sub1 contains a virtual machine named VM1 and a storage account named storage1.
VM1 is associated to the resources shown in the following table.
You need to move VM1 to Sub2.
Which resources should you move to Sub2?

  • A..VM1, Disk1, and NetInt1 only
  • B..VM1, Disk1, and VNet1 only
  • C..VM1, Disk1, and storage1 only
  • CorrectD..VM1, Disk1, NetInt1, and VNet1
Suggested Answer:
D..VM1, Disk1, NetInt1, and VNet1 — If all resources (including the VNet, after making it empty) can be moved together, then option D is correct because it includes all resources associated with VM1 and follows Azure’s move requirements.
-----------------
A..VM1, Disk1, and NetInt1 only — This is correct only if you are not moving the VNet.
B..VM1, Disk1, and VNet1 only — The network interface (NetInt1) must also be moved with the VM.
C..VM1, Disk1, and storage1 only — Storage1 is not required unless it contains unattached data disks; NetInt1 and possibly VNet1 are required.
