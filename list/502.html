You have an Azure subscription.

You create a routing table named RT1.
You need to add a route to RT1 that specifies the next hop IP address.
Which next hop type should you select?

  • A..Internet
  • B..Virtual network gateway
  • C..Virtual network
  • CorrectD..Virtual appliance
Suggested Answer:
D..Virtual appliance — The "Virtual appliance" next hop type allows you to specify a custom next hop IP address, typically used for network virtual appliances such as firewalls or routers.
-----------------
A..Internet — This sends traffic directly to the Internet, not to a specific IP.
B..Virtual network gateway — This routes traffic to an Azure VPN gateway, not a specific IP address.
C..Virtual network — This routes traffic within the VNet, not to a custom IP.
