-----------------
Additional:
Same Subscription: Yes, VNets in the same subscription can be peered directly.
Different Subscriptions: Yes, VNets in different subscriptions can also be peered as long as they are in the same Azure Active Directory (AAD) tenant (Microsoft Entra tenant). This is known as cross-subscription peering.
Key Point: If VNets are in different subscriptions but in the same Entra tenant, peering is allowed. If the subscriptions are linked to different Entra tenants, peering is not allowed.
