Question : 134 Topic: 5

You create an Azure VM named VM1 that runs Windows Server 2019.

VM1 is configured as shown in the exhibit. (Click the Exhibit tab.)
You need to enable Desired State Configuration for VM1.
What should you do first?

  • A..Connect to VM1.
  • CorrectB..Start VM1.
  • C..Capture a snapshot of VM1.
  • D..Configure a DNS name for VM1.
Suggested Answer:
B.. Start VM1
— The VM (VM1) is currently in a Stopped (deallocated) state, as shown in the exhibit. To enable Desired State Configuration (DSC), the VM must be running, so the first step is to start the VM.
-----------------
A.. Connect to VM1 — You cannot connect to a VM that is stopped/deallocated. The VM must be started first.
C.. Capture a snapshot of VM1 — Taking a snapshot is not required to enable DSC and cannot be done while the VM is deallocated.
D.. Configure a DNS name for VM1 — Configuring a DNS name is unrelated to enabling DSC and is not required as the first step.
