Question : 11 Topic: 2

You have an Azure Active Directory (Azure AD) tenant named contoso.onmicrosoft.com that contains 100 user accounts.

You purchase 10 Azure AD Premium P2 licenses for the tenant.
You need to ensure that 10 users can use all the Azure AD Premium features.
What should you do?

  • CorrectA..From the Licenses blade of Azure AD, assign a license
  • B..From the Groups blade of each user, invite the users to a group
  • C..From the Azure AD domain, add an enterprise application
  • D..From the Directory role blade of each user, modify the directory role
Suggested Answer:

Question 2-11 Answer:

A. From the Licenses blade of Azure AD, assign a license
-----------------

Explanation of Correct Option:

A. From the Licenses blade of Azure AD, assign a license — This is the correct method to allocate Azure AD Premium P2 licenses to specific users. Only users who are assigned a valid license can access the premium features such as Identity Protection, Privileged Identity Management, and Conditional Access.
-----------------

Explanation of Other Options:

B. From the Groups blade of each user, invite the users to a group — Incorrect because group membership alone does not assign licenses unless the group is configured for license assignment.
C. From the Azure AD domain, add an enterprise application — Incorrect because enterprise applications are unrelated to license assignment.
D. From the Directory role blade of each user, modify the directory role — Incorrect because assigning a directory role does not grant access to premium features unless the user also has the appropriate license.

How to Assign Azure AD Premium Licenses to Users:

1. Sign in to the Azure portal as a global administrator.
2. Navigate to Azure Active Directory > Licenses.
3. Select All products, then click on the Azure AD Premium P2 license.
4. Click Assign.
5. Choose Users and groups, then select the 10 users you want to assign the license to.
6. Click Assign to complete the process.
-----------------

Notes:

- Only users with assigned licenses will have access to Azure AD Premium features. - You can also assign licenses via groups if group-based licensing is configured.
