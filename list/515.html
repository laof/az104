DRAG DROP

-
You have an Azure subscription that contains a resource group named RG1.
You plan to create an Azure Resource Manager (ARM) template to deploy a new virtual machine named VM1. VM1 must support the capture of performance data.
You need to specify resource dependencies for the ARM template.
In which order should you deploy the resources? To answer, move all resources from the list of resources to the answer area and arrange them in the correct order.

Suggested Answer:
virtual network → network interface → virtual machine → Azure Monitor extension
-----------------

Explanation of Right Options

virtual network must be created first, as network interfaces require a virtual network to exist.
network interface comes next, as a virtual machine requires a network interface to be attached.
virtual machine is then created and attached to the network interface.
Azure Monitor extension is deployed last, as it is an extension that is installed on the running virtual machine.

Explanation of Other Options

Any other order would violate resource dependencies: for example, creating the network interface before the virtual network would not work, nor could the Azure Monitor extension be deployed before the VM exists.
