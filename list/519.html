Question : 146 Topic: 5

You have an Azure subscription.

You plan to migrate 50 virtual machines from VMware vSphere to the subscription.
You create a Recovery Services vault.
What should you do next?

  • A..Configure an extended network.
  • B..Create a recovery plan.
  • C..Deploy an Open Virtualization Application (OVA) template to vSphere.
  • CorrectD..Configure a virtual network.
Suggested Answer:
D..Configure a virtual network.
-----------------

Explanation of Right Options

D..Configure a virtual network. According to the official Microsoft documentation for preparing Azure for disaster recovery of on-premises VMware VMs (https://learn.microsoft.com/en-us/azure/site-recovery/tutorial-prepare-azure), after you create a Recovery Services vault, the next step is to configure a virtual network in Azure. This network is required as the target for your migrated virtual machines.

Explanation of Other Options

A..Configure an extended network. This is not the immediate next step after creating the Recovery Services vault.
B..Create a recovery plan. Recovery plans are created after setting up replication and resources.
C..Deploy an Open Virtualization Application (OVA) template to vSphere. Deploying the OVA template is done later, after Azure-side resources (like the network and storage) are ready.
