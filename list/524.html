You have an Azure subscription that has a Recovery Services vault named Vault1. The subscription contains the virtual machines shown in the following table:

You plan to schedule backups to occur every night at 23:00.
Which virtual machines can you back up by using Azure Backup?

  • A..VM1 and VM3 only
  • CorrectB..VM1, VM2, VM3 and VM4
  • C..VM1 and VM2 only
  • D..VM1 only
Suggested Answer:
B..VM1, VM2, VM3 and VM4
-----------------

Explanation of Right Options

B..VM1, VM2, VM3 and VM4 — Azure Backup supports backing up all listed VM operating systems (Windows Server, Ubuntu Linux, Windows 10), and auto-shutdown does not prevent scheduled backups from happening. The scheduled backup at 23:00 can occur regardless of the auto-shutdown configuration.

Explanation of Other Options

A..VM1 and VM3 only — Incorrect because Azure Backup supports all OS types listed, not just these two.
C..VM1 and VM2 only — Incorrect because both Linux and Windows 10 are also supported.
D..VM1 only — Incorrect, as this unnecessarily excludes other supported VMs.
