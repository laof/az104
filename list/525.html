Question : 2 Topic: 6

You have an Azure subscription that contains a virtual machine named VM1.

You plan to deploy an Azure Monitor alert rule that will trigger an alert when CPU usage on VM1 exceeds 80 percent.
You need to ensure that the alert rule sends an email message to two users named User1 and User2.
What should you create for Azure Monitor?

  • CorrectA..an action group
  • B..a mail-enabled security group
  • C..a distribution group
  • D..a Microsoft 365 group
Suggested Answer:
A..an action group
-----------------

Explanation of Right Options

A..an action group — In Azure Monitor, action groups are used to define a set of actions, such as sending email notifications, when an alert is triggered. You can specify multiple recipients (such as User1 and User2) in an action group to ensure both receive the alert notification.

Explanation of Other Options

B..a mail-enabled security group — Mail-enabled security groups are for access control and not directly used in Azure Monitor alert notifications.
C..a distribution group — Distribution groups are for email distribution lists in Exchange/Outlook, not for configuring alert actions in Azure Monitor.
D..a Microsoft 365 group — Microsoft 365 groups are used for collaboration in Microsoft 365 services, not for Azure Monitor alert notification configuration.
