Question : 13 Topic: 2

You sign up for Azure Active Directory (Azure AD) Premium P2.

You need to add a user named [email protected] as an administrator on all the computers that will be joined to the Azure AD domain.
What should you configure in Azure AD?

  • CorrectA..Device settings from the Devices blade
  • B..Providers from the MFA Server blade
  • C..User settings from the Users blade
  • D..General settings from the Groups blade
Suggested Answer:

Question 2-13 Answer:

A. Device settings from the Devices blade
-----------------

Explanation of Correct Option:

A. Device settings from the Devices blade — This is where you configure the option to add additional users as local administrators on Azure AD-joined devices. By setting [email protected] as an additional local admin in the Device settings, they will automatically be granted admin rights on all devices joined to Azure AD.
-----------------

Explanation of Other Options:

B. Providers from the MFA Server blade — Incorrect because this is related to multi-factor authentication, not device administration.
C. User settings from the Users blade — Incorrect because this controls user-level settings like external collaboration, not device admin roles.
D. General settings from the Groups blade — Incorrect because group settings do not control device administrator roles.

Example Process to Add a User as Local Administrator on Azure AD-Joined Devices:

1. Sign in to the Azure portal as a Global Administrator.
2. Navigate to Azure Active Directory > Devices.
3. Click on Device settings.
4. Under Additional local administrators on Azure AD joined devices, click Selected.
5. Click + Add members and select the user (e.g., [email protected]).
6. Click Select, then Save to apply the changes.
-----------------

Notes:

- This setting ensures the selected user is automatically added as a local administrator on all Azure AD-joined devices.
- Changes apply to future device joins and may require reauthentication on existing devices.
