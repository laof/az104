Suggested Answer:
1. After User2 answers three security questions correctly, he can reset his password immediately. No
User2 is a member of Group2, and self-service password reset is enabled for Group2 (see Image1). The authentication configuration requires two methods to reset the password (see Image2). Answering three security questions satisfies only one method, so User2 must also verify with the Mobile phone method to complete the reset processâ€”answering security questions alone is not enough.
2. If User1 forgets her password, she can reset the password by using the mobile phone app. No
User1 is a member of Group1 only, and only Group2 is enabled for self-service password reset (see Image1). Therefore, User1 cannot reset her password using any method, including mobile phone app.
3. User3 can add security questions to the password reset process. Yes
User3 is a User administrator and a member of both Group1 and Group2. As an admin, User3 can manage and configure security questions for the password reset process for the organization, even though admins themselves cannot use security questions for their own password reset authentication (see information in Image2). The statement is about adding (configuring) security questions, not using them personally.
