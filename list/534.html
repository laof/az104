HOTSPOT -

You have two Azure App Service app named App1 and App2. Each app has a production deployment slot and a test deployment slot.
The Backup Configuration settings for the production slots are shown in the following table.
For each of the following statements, select Yes if the statement is true. Otherwise, select No.
NOTE: Each correct selection is worth one point.
Hot Area:

Suggested Answer:
On January 15, 2021, App1 will have only one backup in storage: Yes
App1 is backed up daily starting January 6, 2021, with retention set to 0 days, but with "Keep at least one backup" enabled. This means only the most recent backup is kept, so on January 15, only the backup from that day will be in storage.
On February 6, 2021, you can access the backup of the App2 test slot from January 15, 2021: No
The backup configuration applies only to the production slots, not the test slots. There is no indication that the App2 test slot has its own backup or retention policy.
On January 15, 2021, you can restore the App2 production slot backup from January 6 to the App2 test slot: Yes
App2's production slot backups are retained for 30 days, so the January 6 backup will still be available on January 15. Azure App Service allows restoring a production slot backup to a test slot.
