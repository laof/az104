You have an Azure subscription that contains the following users in an Azure Active Directory tenant named contoso.onmicrosoft.com:
User1 creates a new Azure Active Directory tenant named external.contoso.onmicrosoft.com.
You need to create new user accounts in external.contoso.onmicrosoft.com.
Solution: You instruct User1 to create the user accounts.
Does that meet the goal?

  • CorrectA..Yes
  • B..No
Suggested Answer:
A..Yes

Explanation of Right Option

When User1 creates a new Azure Active Directory tenant (external.contoso.onmicrosoft.com), they are automatically assigned the Global administrator role in that tenant. As a Global administrator, User1 has full permissions to create new user accounts in the new tenant, so instructing User1 to create user accounts does meet the goal.
-----------------
B..No This is incorrect because User1, as the Global administrator of the new tenant, has all the necessary rights to create user accounts. No additional role assignment or configuration is required for this permission.
