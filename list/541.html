Question : 18 Topic: 6

You have an Azure virtual machine named VM1.

Azure collects events from VM1.
You are creating an alert rule in Azure Monitor to notify an administrator when an error is logged in the System event log of VM1.
Which target resource should you monitor in the alert rule?

  • A..virtual machine extension
  • B..virtual machine
  • C..metric alert
  • CorrectD..Azure Log Analytics workspace
Suggested Answer:
D..Azure Log Analytics workspace

Explanation of Right Option

Azure Log Analytics workspace should be selected as the target resource because when creating a log-based alert that monitors the System event log from a VM, the event data is collected and stored within the Log Analytics workspace. The alert rule evaluates the logs in this workspace, not directly on the VM.
-----------------
B..virtual machine This would be correct for metric-based alerts (such as CPU or memory usage), but not for log-based alerts that rely on data in Log Analytics.
A..virtual machine extension Extensions are used to install agents or provide additional configuration, not for alert targeting.
C..metric alert This is not a target resource but a type of alert for performance metrics, not event logs.
