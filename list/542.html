Question : 19 Topic: 6

You have an Azure subscription that contains 100 virtual machines.

You regularly create and delete virtual machines.
You need to identify unattached disks that can be deleted.
What should you do?

  • A..From Azure Cost Management, view Cost Analysis
  • B..From Azure Advisor, modify the Advisor configuration
  • C..From Microsoft Azure Storage Explorer, view the Account Management properties
  • CorrectD..From Azure Cost Management, view Advisor Recommendations
Suggested Answer:
D..From Azure Cost Management, view Advisor Recommendations

Explanation of Right Option

Advisor Recommendations (accessible from Azure Cost Management) will provide cost-saving suggestions, including identifying unattached (orphaned) disks that can be safely deleted. This is the recommended and most efficient way to identify such resources.
-----------------
A..From Azure Cost Management, view Cost Analysis This shows cost data but does not identify unattached disks.
B..From Azure Advisor, modify the Advisor configuration Modifying the configuration is not how to view recommendations; you need to view recommendations directly.
C..From Microsoft Azure Storage Explorer, view the Account Management properties This tool is used for managing storage accounts, not for listing unattached VM disks.
