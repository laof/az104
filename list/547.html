HOTSPOT -

You have an Azure subscription that contains the resources shown in the following table.
You plan to create a data collection rule named DCR1 in Azure Monitor.
Which resources can you set as data sources in DCR1, and which resources can you set as destinations in DCR1? To answer, select the appropriate options in the answer area.
NOTE: Each correct selection is worth one point.
Hot Area:

Suggested Answer:
Data sources: VM1 only
Destinations: Workspace1 and storage1 only

Explanation of Right Options

VM1 only: For Azure Monitor Data Collection Rules (DCR), virtual machines (like VM1) are supported as data sources for collecting telemetry and performance data. Other resources in the list (storage1, Workspace1, DB1) cannot serve as data sources for DCR.
Workspace1 and storage1 only: Log Analytics workspaces (Workspace1) and storage accounts (storage1) can be set as destinations in DCR, allowing collected data to be sent to these resources for analysis and archival. DB1 (Azure SQL database) is not a valid DCR destination.
-----------------
Data sources: VM1 and storage1 only / VM1, storage1, and DB1 only / VM1, storage1, Workspace1, and DB1
storage1, Workspace1, and DB1 cannot serve as data sources for DCR; only VMs can be data sources.
Destinations: storage1 only / Workspace1 only / Workspace1, storage1, and DB1 only
DB1 is not a supported DCR destination, and setting only storage1 or only Workspace1 misses one valid destination (either storage or workspace).
