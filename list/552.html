HOTSPOT -

You have two Azure subscriptions named Sub1 and Sub2. Sub1 is in a management group named MG1. Sub2 is in a management group named MG2.
You have the resource groups shown in the following table.
You have the virtual machines shown in the following table.
You assign roles to users as shown in the following table.
For each of the following statements, select Yes if the statement is true. Otherwise, select No.
NOTE: Each correct selection is worth one point.
Hot Area:

Suggested Answer:
A..No
B..No
C..Yes

Explanations of Right Options

C..User2 can manage disks and disk snapshots of VM3.: User2 has the Virtual Machine Contributor role assigned at MG2, which covers Sub2 and RG2 (where VM3 resides). This role allows management of VM resources, including disks and snapshots.
-----------------
A..User1 can sign in to VM1.: User1 has Virtual Machine Contributor at MG1 (covering VM1), but this role does not grant sign-in rights. The Virtual Machine User Login role is required to sign in, and User1 does not have it for MG1, Sub1, RG1, or VM1.
B..User2 can manage disks and disk snapshots of VM1.: User2 has Virtual Machine Contributor at MG2, but VM1 is in Sub1 under MG1. User2 does not have permissions over VM1 or its disks.
