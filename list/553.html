Question : 30 Topic: 6

You have an Azure Active Directory (Azure AD) tenant that is linked to 10 Azure subscriptions.

You need to centrally monitor user activity across all the subscriptions.
What should you use?

  • A..Azure Application Insights Profiler
  • B..access reviews
  • C..Activity log filters
  • CorrectD..a Log Analytics workspace
Suggested Answer:
D..a Log Analytics workspace

Explanations of Right Options

D..a Log Analytics workspace: A Log Analytics workspace in Azure Monitor allows you to collect and analyze activity logs from multiple Azure subscriptions in one central location. By configuring all subscriptions to send their activity logs to a single Log Analytics workspace, you can centrally monitor user activity across all subscriptions.
-----------------
A..Azure Application Insights Profiler: This tool is used for profiling application performance, not for monitoring user activity or activity logs across subscriptions.
B..access reviews: Access reviews are used to periodically review user access to resources, not for centralized monitoring of user activity.
C..Activity log filters: Activity log filters can filter logs within a subscription, but do not provide centralized monitoring across multiple subscriptions.
