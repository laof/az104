DRAG DROP -

You have an Azure subscription that contains a virtual machine name VM1.
VM1 has an operating system disk named Disk1 and a data disk named Disk2.
You need to back up Disk2 by using Azure Backup.
Which three actions should you perform in sequence? To answer, move the appropriate actions from the list of actions to the answer area and arrange them in the correct order.
Select and Place:

Suggested Answer:
1. Create an Azure Backup vault
2. Create a backup policy and configure the backup
3. Configure a managed identity
-----------------
Explanation of Right Options:
1. Create an Azure Backup vault: This is the essential first step, as Azure disks must be backed up to an Azure Backup vault.
2. Create a backup policy and configure the backup: Next, you define how the backup will operate (schedule, retention, etc.) and set up the backup configuration for Disk2, which prepares the backup process.
3. Configure a managed identity: After the backup setup is defined, you configure a managed identity for the VM/resource so it can securely access the vault and perform backup actions as required.
-----------------
Explanation of Other Options:
Origin: Create a Recovery Services vault: Incorrect for this scenario, as Recovery Services vaults do not support standalone Azure disk backups.
Origin: Delegate permissions for the vault: Not needed as a separate step here, since managed identity configuration will handle necessary access for backup operations.
