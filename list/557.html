Question : 34 Topic: 6

You have an Azure App Service web app named App1.

You need to collect performance traces for App1.
What should you use?

  • CorrectA..Azure Application Insights Profiler
  • B..the Activity log
  • C..the Deployment center
  • D..the Diagnose and solve problems settings
Suggested Answer:<
A. Azure Application Insights Profiler: The Azure Application Insights Profiler is the correct tool to collect detailed performance traces for an Azure App Service web app like App1. It enables you to gather in-depth performance data, call stacks, and trace information to help diagnose and optimize your application's performance.
-----------------
Explanation of Other Options:
the Activity log: The Activity log records management events (like resource creation or deletion) in Azure, but it does not provide application-level performance tracing.
the Deployment center: The Deployment center is used for configuring and managing code deployments to App Service, not for collecting performance traces.
the Diagnose and solve problems settings: While this feature offers troubleshooting and diagnostics tools, it does not specifically provide the deep performance tracing that the Application Insights Profiler does.
