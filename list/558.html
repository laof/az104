Question : 35 Topic: 6

You have an Azure subscription that contains the storage accounts shown in the following table.

You deploy a web app named App1 to the West US Azure region.
You need to back up App1. The solution must minimize costs.
Which storage account should you use as the target for the backup?

  • A..storage1
  • CorrectB..storage2
  • C..storage3
  • D..storage4
Suggested Answer:
C.. storage3: You should use storage3 as the backup target because it is a BlockBlobStorage account located in West US, the same region as App1. Using a storage account in the same region minimizes costs by avoiding cross-region data transfer charges, and BlockBlobStorage is cost-effective and optimized for backup scenarios.
-----------------
Explanation of Other Options:
A.. storage1: This is a StorageV2 account but is in Central US. Choosing a storage account in a different region increases data transfer costs.
B.. storage2: This is a BlobStorage account in West US, but BlockBlobStorage (storage3) is more optimized and cost-efficient for backups than general BlobStorage.
D.. storage4: This is a FileStorage account in East US, which is both the wrong storage type and region for a cost-effective, blob-based backup.
Additional:
Blobstorage is cheaper than the filestorage and StorageV2
