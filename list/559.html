HOTSPOT -

You have an Azure subscription that is linked to an Azure AD tenant. The tenant contains two users named User1 and User2.
The subscription contains the resources shown in the following table.
The subscription contains the alert rules shown in the following table.
The users perform the following action:
• User1 creates a new virtual disk and attaches the disk to VM1
• User2 creates a new resource tag and assigns the tag to RG1 and VM1
Which alert rules are triggered by each user? To answer, select the appropriate options in the answer area.
NOTE: Each correct selection is worth one point.

Suggested Answer:
User1: Alert1 and Alert2 are triggered
User2: Alert1 and Alert2 are triggered
-----------------
Explanation of Right Options:
User1: Creating and attaching a new virtual disk to VM1 is an administrative operation on VM1, which triggers Alert2 (scope: VM1, condition: all administrative operations); since VM1 is in RG1, and Alert1 (scope: RG1) applies to administrative operations for any resource in the group, Alert1 is also triggered.
User2: Assigning a tag to both RG1 and VM1 is an administrative operation for each resource, so tagging RG1 triggers Alert1, and tagging VM1 triggers both Alert1 (because it's in RG1) and Alert2 (scope: VM1), so both alerts are triggered.
-----------------
Explanation of Other Options:
No alert is triggered: Incorrect, because both users perform administrative operations that match the scope and condition of at least one alert.
Only Alert1 is triggered: Incorrect, because actions on VM1 also trigger Alert2 in addition to Alert1.
Only Alert2 is triggered: Incorrect, because administrative operations on VM1 and RG1 both fall under Alert1's scope, so both alerts are triggered.
