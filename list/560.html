Question : 37 Topic: 6

You have an Azure subscription that contains multiple virtual machines in the West US Azure region.

You need to use Traffic Analytics in Azure Network Watcher to monitor virtual machine traffic.
Which two resources should you create? Each correct answer presents part of the solution.
NOTE: Each correct selection is worth one point.

  • CorrectA..a Log Analytics workspace
  • B..an Azure Monitor workbook
  • CorrectC..a storage account
  • D..a Microsoft Sentinel workspace
  • E..a Data Collection Rule (DCR) in Azure Monitor
Suggested Answer:
A: a Log Analytics workspace
C: a storage account
-----------------
Explanation of Right Options:
Log Analytics workspace (A): Traffic Analytics in Azure Network Watcher requires a Log Analytics workspace to store and analyze the flow logs and traffic data.
Storage account (C): Network Security Group (NSG) flow logs must be stored in an Azure storage account before they are analyzed by Traffic Analytics.
-----------------
Explanation of Other Options:
Azure Monitor workbook (B): This is used for visualizing and reporting, but is not required to enable Traffic Analytics.
Microsoft Sentinel workspace (D): This is a SIEM solution and not required for Traffic Analytics.
Data Collection Rule (DCR) in Azure Monitor (E): DCR is used for data collection in Azure Monitor but is not part of the required setup for Traffic Analytics with Network Watcher.
