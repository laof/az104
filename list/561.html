Question : 38 Topic: 6

You have an Azure subscription that contains eight virtual machines and the resources shown in the following table.

You need to configure access for VNET1. The solution must meet the following requirements:
• The virtual machines connected to VNET1 must be able to communicate with the virtual machines connected to VNET2 by using the Microsoft backbone.
• The virtual machines connected to VNET1 must be able to access storage1, storage2, and Azure AD by using the Microsoft backbone.
What is the minimum number of service endpoints you should add to VNET1?

  • A..1
  • CorrectB..2
  • C..3
  • D..5
Suggested Answer:
B. 2
-----------------
Explanation of Right Options:
To allow VNET1 VMs to access storage1 and storage2 using the Microsoft backbone, you need a Microsoft.Storage service endpoint. To access Azure AD, you need a Microsoft.AzureActiveDirectory service endpoint. Communication between VNET1 and VNET2 can be achieved with VNet peering and does not require a service endpoint. Therefore, the minimum number of service endpoints required is 2: one for Storage and one for Azure AD.
-----------------
Explanation of Other Options:
1: Incorrect, because you need to enable both Storage and Azure AD endpoints for the listed requirements.
3: Incorrect, because only two service endpoints are needed (not for Key Vault or other services mentioned).
5: Incorrect, as there is no requirement for five different endpoint types in this scenario.
