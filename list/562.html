Question : 39 Topic: 6

You need to configure an Azure web app named contoso.azurewebsites.net to host www.contoso.com.

What should you do first?

  • A..Create A records named www.contoso.com and asuid.contoso.com.
  • CorrectB..Create a TXT record named asuid that contains the domain verification ID.
  • C..Create a CNAME record named asuid that contains the domain verification ID.
  • D..Create a TXT record named www.contoso.com that has a value of contoso.azurewebsites.net.
Suggested Answer:
B. Create a TXT record named asuid that contains the domain verification ID.
-----------------
Explanation of Right Options:
The first step to configure a custom domain for an Azure web app is to create a TXT record named asuid with the domain verification ID provided by Azure. This TXT record is used by Azure to verify domain ownership before you can map the domain to your web app, as shown in the provided documentation.
-----------------
Explanation of Other Options:
A: Creating A records named www.contoso.com and asuid.contoso.com is not the required first step for domain verification in Azure Web Apps.
C: Creating a CNAME record named asuid with the domain verification ID is not the correct method for verification.
D: Creating a TXT record named www.contoso.com with a value of contoso.azurewebsites.net is not used for domain ownership verification.
Additional:
Before you can point your domain to an Azure Web App, you need to verify that you actually own the domain.
1. Obtain Domain Verification ID
In the Azure portal, navigate to your Azure Web App.
Go to the "Custom domains" section.
Azure provides a unique domain verification ID.
2. Create the TXT Record
Log in to your domain registrar's (e.g., GoDaddy, Namecheap) DNS management area.
Add a new TXT record.
The name of this record will be "asuid".
The value of the record should be the domain verification ID you got from Azure.
3. Add CNAME Record (Later)
Once Azure has verified that you own the domain, you'll then need to create a CNAME record named "www" that points to "contoso.azurewebsites.net". This tells the DNS system where to direct web traffic for www.contoso.com.
