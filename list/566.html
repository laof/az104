You have an Azure subscription that contains two Log Analytics workspaces named Workspace1 and Workspace2 and 100 virtual machines that run Windows Server.

You need to collect performance data and events from the virtual machines. The solution must meet the following requirements:
• Logs must be sent to Workspace1 and Workspace 2.
• All Windows events must be captured.
• All security events must be captured.
What should you install and configure on each virtual machine?

  • CorrectA..the Azure Monitor agent
  • B..the Windows Azure diagnostics extension (WAD)
  • C..the Windows VM agent
Suggested Answer:
A. the Azure Monitor agent
Explanation: The Azure Monitor agent supports collecting performance data, all Windows events, all security events, and sending logs to multiple Log Analytics workspaces, fulfilling all requirements with minimal configuration.
-----------------
B. Windows Azure diagnostics extension (WAD): Does not support sending logs to multiple workspaces and is not recommended for new solutions.
C. Windows VM agent: Is required for installing extensions but does not collect or send data itself; it cannot fulfill the requirements alone.
