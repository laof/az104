Question : 44 Topic: 6

You have an Azure subscription that contains a virtual machine named VM1 and an Azure function named App1.

You need to create an alert rule that will run App1 if VM1 stops.
What should you create for the alert rule?

  • A..an application security group
  • B..a security group that has dynamic device membership
  • CorrectC..an action group
  • D..an application group
Suggested Answer:
C. an action group
Explanation: An action group in Azure is a collection of notification preferences and actions that can be used by alert rules. To run an Azure function (such as App1) in response to an alert (such as when VM1 stops), you must configure the alert rule to trigger an action group, and add the Azure Function as an action within that action group.
-----------------
A. application security group: Used to group VMs for network security management, not for alerting or automation.
B. security group with dynamic device membership: Related to Azure AD for managing access/control, not for alerts or triggering actions.
D. application group: Used with Azure Virtual Desktop to manage app assignments, not for alerts or automation.
