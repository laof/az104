Question : 47 Topic: 6

You have an Azure subscription.

You need to receive an email alert when a resource lock is removed from any resource in the subscription.
What should you use to create an activity log alert in Azure Monitor?

  • CorrectA..a resource, a condition, and an action group
  • B..a resource, a condition, and a Microsoft 365 group
  • C..a Log Analytics workspace, a resource, and an action group
  • D..a data collection endpoint, an application security group, and a resource group
Suggested Answer:
A. a resource, a condition, and an action group
Explanation: To create an activity log alert in Azure Monitor for resource lock removals, you must define a resource (scope for the alert), a condition (what operation/event triggers the alert), and an action group (how notifications, such as email, are sent).
-----------------
B. a resource, a condition, and a Microsoft 365 group: Microsoft 365 groups are not used for Azure Monitor alert notifications; action groups are required.
C. a Log Analytics workspace, a resource, and an action group: Log Analytics workspaces are for log data analysis, not a requirement for activity log alerts.
D. a data collection endpoint, an application security group, and a resource group: These are unrelated to alerting on resource lock events in Azure Monitor.
