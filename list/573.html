HOTSPOT

-
You have an Azure subscription that contains the vaults shown in the following table.
You deploy the virtual machines shown in the following table.
You have the backup policies shown in the following table.
For each of the following statements, select Yes if the statement is true. Otherwise, select No.
NOTE: Each correct selection is worth one point.

Suggested Answer:

Correct Option Explanation

VM1 can be backed up by using Policy1: Yes
Explanation: VM1 is Windows Server, and Policy1 is a Standard policy in Recovery1 (a Recovery Services vault), which supports Windows Server VM backup.
VM2 can be backed up by using Policy3: Yes
Explanation: VM2 is Linux, and Policy3 is in Backup1 (an Azure Backup vault) where the backup policy type is managed by the vault and applies to VMs, so VM2 can be backed up by Policy3.
VM2 can be backed up by using Policy2: No
Explanation: Policy2 is in Recovery2, which does not exist in the available vaults, so Policy2 cannot be used for VM2.
Additional:
Azure Backup is not limited to VMs. It supports a wide range of data sources, including:
- Azure VMs (Windows/Linux)
- Azure SQL databases
- Azure Files
- Azure Blob storage
- Microsoft SQL Server
- Microsoft Exchange
- Microsoft SharePoint
- Virtual machine disks (via backup policies)
- On-premises servers (via Azure Backup Server)
