Question : 53 Topic: 6

You have an Azure subscription that contains an Azure Stream Analytics job named Job1.

You need to monitor input events for Job1 to identify the number of events that were NOT processed.
Which metric should you use?

  • A..Out-of-Order Events
  • B..Output Events
  • C..Late Input Events
  • CorrectD..Backlogged Input Events
Suggested Answer:
D. Backlogged Input Events
Explanation: The Backlogged Input Events metric shows the number of input events that have not yet been processed by the Stream Analytics job, helping you identify events that were NOT processed.
-----------------

Other Options Explanation

A. Out-of-Order Events: This metric counts events arriving out of order, not those that are unprocessed.
B. Output Events: This metric shows the number of events successfully output by the job, not unprocessed events.
C. Late Input Events: This metric measures events that arrived after their window closed, but does not represent all unprocessed events.
