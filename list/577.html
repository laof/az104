Question : 54 Topic: 6

You have an Azure subscription that contains an Azure SQL database named DB1.

You plan to use Azure Monitor to monitor the performance of DB1. You must be able to run queries to analyze log data.
Which destination should you configure in the Diagnostic settings of DB1?

  • CorrectA..Send to a Log Analytics workspace.
  • B..Archive to a storage account.
  • C..Stream to an Azure event hub.
Suggested Answer:
A. Send to a Log Analytics workspace.
Explanation: To monitor the performance of DB1 and run queries to analyze log data, you should send diagnostic logs to a Log Analytics workspace. This destination enables advanced querying and analysis using Kusto Query Language (KQL).
-----------------

Other Options Explanation

B. Archive to a storage account: This stores logs for long-term retention, but does not provide built-in query and analysis capabilities.
C. Stream to an Azure event hub: This is used for integration with external systems or event processing, but is not designed for querying or analysis of log data.
