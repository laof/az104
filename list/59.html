You have an Azure Directory (Azure AD) tenant named Adatum and an Azure Subscription named Subscription1. Adatum contains a group named Developers.
Subscription1 contains a resource group named Dev.
You need to provide the Developers group with the ability to create Azure logic apps in the Dev resource group.
Solution: On Subscription1, you assign the Logic App Operator role to the Developers group.
Does this meet the goal?

  • A..Yes
  • CorrectB..No
Suggested Answer:

Question 2-19

Correct Answer: B..No
Explanation: The Logic App Operator role allows users to read, enable, disable, and run logic apps, but it does not grant permissions to create or edit logic apps. To meet the goal of allowing the Developers group to create logic apps, a role like Logic App Contributor or Contributor is required.
-----------------
A..Yes: Incorrect. The Logic App Operator role does not include permissions to create or modify logic apps, so it does not meet the goal.
B..No: Correct. Since the assigned role does not allow creation of logic apps, the solution does not meet the goal.
