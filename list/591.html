You need to implement a backup solution for App1 after the application is moved.
What should you create first?

  • A..a recovery plan
  • B..an Azure Backup Server
  • C..a backup policy
  • CorrectD..a Recovery Services vault
Suggested Answer:
D. a Recovery Services vault
Explanation:
A Recovery Services vault is the first thing you must create in Azure to protect virtual machines with backups; the vault acts as the storage and management container for backup data and backup policies, and is required before configuring backup protection for the VMs of App1 after migration.
Explanation of Other Options
A. a recovery plan: Recovery plans are designed for orchestrating disaster recovery with Azure Site Recovery, not for configuring backups for Azure VMs.
B. an Azure Backup Server: Azure Backup Server is for on-premises backups and is not required when protecting Azure VMs with native Azure Backup.
C. a backup policy: Backup policies define the backup schedule and retention, but you must have a Recovery Services vault first to store and apply the policy to the VMs.
