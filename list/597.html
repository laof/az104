Scenario

Must meet the technical requirements including:
- Ensure VM3 can establish outbound connections over TCP port 8080 to application servers in the Montreal office.
You discover that VM3 does NOT meet the technical requirements.
You need to verify whether the issue relates to the NSGs.
What should you use?

  • A..Diagram in VNet1
  • B..Diagnostic settings in Azure Monitor
  • C..Diagnose and solve problems in Traffic Manager profiles
  • D..The security recommendations in Azure Advisor
  • CorrectE..IP flow verify in Azure Network Watcher
Suggested Answer:
E. IP flow verify in Azure Network Watcher.
Explanation: IP flow verify in Azure Network Watcher is the best tool to check if NSG rules are allowing or blocking the required traffic, as it tests specific traffic flows based on source, destination, port, and protocol, directly identifying NSG-related issues.
Other options like VNet diagrams, Azure Monitor diagnostics, Traffic Manager troubleshooting, and Azure Advisor recommendations do not directly verify NSG rule effects on specific traffic flows.
