Question : 2 Topic: 16

Technical Requirement

- Ensure Azure Multi-Factor Authentication(MFA) for the user in the finance department only.
You need to recommend a solution to automate the configuration for the finance department users. The solution must meet the technical requirements.
What should you include in the recommendation?

  • A..Azure AD B2C
  • CorrectB..dynamic groups and conditional access policies
  • C..Azure AD Identity Protection
  • D..an Azure logic app and the Microsoft Identity Management (MIM) client
Suggested Answer:
B..dynamic groups and conditional access policies
Explanation of Right Option
Dynamic groups and conditional access policies allow you to automatically assign users to a group based on department (e.g., finance) and then apply a Conditional Access Policy to require Multi-Factor Authentication (MFA) only for that group. This meets the requirement for automating MFA for only finance users.
Explanation of Other Options
Azure AD B2C is for customer-facing identity solutions and is not intended for internal user MFA enforcement.
Azure AD Identity Protection provides automated risk-based conditional access but is not directly used to enforce MFA for a specific department.
An Azure logic app and the Microsoft Identity Management (MIM) client is an overly complex solution for this scenario and is not typically used to enforce MFA for department-based groups.
