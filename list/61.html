DRAG DROP -

You have an Azure subscription that is used by four departments in your company. The subscription contains 10 resource groups. Each department uses resources in several resource groups.
You need to send a report to the finance department. The report must detail the costs for each department.
Which three actions should you perform in sequence? To answer, move the appropriate actions from the list of actions to the answer area and arrange them in the correct order.
Select and Place:

Suggested Answer:

Question 2-21

Correct Answer:
1. Assign a tag to each resource
2. From the Cost analysis blade, filter the view by tag
3. Download the usage report
Explanation: To report costs by department, you must first tag each resource with department-specific metadata. Then, in the Azure portal, go to the Cost analysis blade and filter by tag to view department-specific costs. Finally, you download the usage report to share or further analyze the data.
-----------------
Assign a tag to each resource group: Incorrect. Tagging resource groups is not sufficient because costs are tracked at the resource level.
Assign a tag to each resource: Correct. This ensures cost data can be associated with specific departments.
Download the usage report: Correct, but should be done after filtering to ensure relevant data is captured.
From the Cost analysis blade, filter the view by tag: Correct. This allows you to isolate and analyze costs by department.
Open the Resource costs blade of each resource group: Incorrect. This does not support tag-based filtering across departments.
