Question : 27 Topic: 2

You have an Azure subscription.

You have 100 Azure virtual machines.
You need to quickly identify underutilized virtual machines that can have their service tier changed to a less expensive offering.
Which blade should you use?

  • A..Monitor
  • CorrectB..Advisor
  • C..Metrics
  • D..Customer insights
Suggested Answer:

Question 2-27

Correct Answer: B..Advisor
Explanation: Azure Advisor provides personalized recommendations to optimize your Azure resources for high availability, security, performance, and cost. It includes cost recommendations such as identifying underutilized virtual machines that can be resized to a lower-cost SKU.
-----------------
A..Monitor: Incorrect. Azure Monitor provides data collection and analysis but does not offer cost-saving recommendations.
B..Advisor: Correct. Advisor analyzes your resource usage and provides recommendations, including identifying underutilized VMs.
C..Metrics: Incorrect. Metrics shows performance data but does not provide optimization suggestions.
D..Customer insights: Incorrect. This is related to customer data and analytics, not virtual machine performance or cost optimization.
