HOTSPOT -

You have an Azure Active Directory (Azure AD) tenant.
You need to create a conditional access policy that requires all users to use multi-factor authentication when they access the Azure portal.
Which three settings should you configure? To answer, select the appropriate settings in the answer area.
NOTE: Each correct selection is worth one point.
Hot Area:

Suggested Answer:

Question 2-28

Correct Answer:
1. Users and groups: Select All users
2. Cloud apps: Select Microsoft Azure Management
3. Grant: Select Grant access and Require multi-factor authentication
Explanation:
To enforce MFA for all users accessing the Azure portal, you must:
- Target all users under Users and groups to ensure the policy applies universally.
- Specify Microsoft Azure Management under Cloud apps to apply the policy only when users access the Azure portal.
- Under Grant, choose Grant access and then select Require multi-factor authentication to enforce MFA.
-----------------
Conditions: Not required in this case unless you want to apply the policy only under specific conditions (e.g., location, device state).
Session: Optional. Used for controlling session behavior but not necessary for enforcing MFA.
