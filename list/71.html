HOTSPOT -

You have an Azure Active Directory (Azure AD) tenant named adatum.com. Adatum.com contains the groups in the following table.
You create two user accounts that are configured as shown in the following table.
Of which groups are User1 and User2 members? To answer, select the appropriate options in the answer area.
NOTE: Each correct selection is worth one point.
Hot Area:

Suggested Answer:

Question 2-31

Correct Answers:
User1: Group1 only
User2: Group1 and Group2 only
Explanation:
Group1: Dynamic security group with rule (user.city -startsWith "m"). Both User1 (Montreal) and User2 (Melbourne) have cities starting with "m", so both are included.
Group2: Dynamic Microsoft 365 group with rule (user.department -notIn ["human resources"]). User1 is in "human resources", so excluded. User2 is in "Marketing", so included.
Group3: Assigned group. Neither user is explicitly assigned, so neither is a member.
-----------------
User1:
- Group1 only: Correct. Matches city rule, but excluded from Group2 due to department.
- Other options: Incorrect because User1 is not in Group2 or Group3.
User2:
- Group1 and Group2 only: Correct. Matches both city and department rules.
- Other options: Incorrect because User2 is not assigned to Group3.
