HOTSPOT -

You have a hybrid deployment of Azure Active Directory (Azure AD) that contains the users shown in the following table.
You need to modify the JobTitle and UsageLocation attributes for the users.
For which users can you modify the attributes from Azure AD? To answer, select the appropriate options in the answer area.
NOTE: Each correct selection is worth one point.
Hot Area:

Suggested Answer:

Correct Answers:

JobTitle: User1 only
UsageLocation: User1,User2,andUser3
-----------------

Explanation of Correct Options:

JobTitle: User1 only
Azure AD cloud-only users (like User1) have their attributes managed directly in Azure AD. For synced users (like User2), attribute changes must be made on-premises and synced. Guest users (like User3) have limited editable attributes.
UsageLocation: User1,User2,andUser3
The UsageLocation attribute can be set in Azure AD for all user types, including cloud-only, synced, and guest users. This is often required for licensing purposes.
-----------------

Explanation of Other Options:

Incorrect because User2 is synced from on-premises AD, and attributes like JobTitle must be edited on-premises, not in Azure AD.
JobTitle: User1 and User3 only
Incorrect because although User3 is a guest, not all attributes like JobTitle are editable for guest accounts in Azure AD.
