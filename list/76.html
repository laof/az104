Question : 36 Topic: 2

You have an Azure subscription that contains a user named User1.

You need to ensure that User1 can deploy virtual machines and manage virtual networks. The solution must use the principle of least privilege.
Which role-based access control (RBAC) role should you assign to User1?

  • A..Owner
  • B..Virtual Machine Contributor
  • CorrectC..Contributor
  • D..Virtual Machine Administrator Login
Suggested Answer:

Question 2-36

Correct Answer: C. Contributor – The Contributor role allows full management of all Azure resources, including the ability to deploy virtual machines and manage virtual networks. It provides the necessary permissions while still adhering to the principle of least privilege better than the Owner role.
-----------------
Explanation of Other Options:
A. Owner – Incorrect. While the Owner role includes all permissions of the Contributor role, it also allows managing access (RBAC), which is more than needed and violates the principle of least privilege.
B. Virtual Machine Contributor – Incorrect. This role allows managing virtual machines but does not include permissions to manage virtual networks, which is part of the requirement.
D. Virtual Machine Administrator Login – Incorrect. This role only allows logging in to virtual machines as an administrator and does not grant permissions to deploy or manage VMs or networks.
