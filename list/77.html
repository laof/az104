HOTSPOT -

You have an Azure Active Directory (Azure AD) tenant that contains three global administrators named Admin1, Admin2, and Admin3.
The tenant is associated to an Azure subscription. Access control for the subscription is configured as shown in the Access control exhibit. (Click the Access
Control tab.)
You sign in to the Azure portal as Admin1 and configure the tenant as shown in the Tenant exhibit. (Click the Tenant tab.)
For each of the following statements, select Yes if the statement is true. Otherwise, select No.
NOTE: Each correct selection is worth one point.
Hot Area:

Suggested Answer:

Question 2-37

Correct Answers:
Admin1 can add Admin2 as an owner of the subscription: Yes – Admin1 has directory-level access management enabled, which allows managing access to all Azure subscriptions and management groups in the directory, even without being explicitly assigned a role at the subscription level.
Admin3 can add Admin2 as an owner of the subscription: Yes – Admin3 is explicitly assigned the Owner role at the subscription level, which includes permission to manage access (RBAC), so Admin3 can assign roles like Owner to others.
Admin2 can create a resource group in the subscription: No – Admin2 is not assigned any role at the subscription level, so they do not have permissions to create or manage resources.
-----------------
Explanation of Other Options:
Admin1 can add Admin2 as an owner of the subscription: No – Incorrect. Admin1 has directory-wide access management enabled, which allows managing access even without a direct role assignment.
Admin3 can add Admin2 as an owner of the subscription: No – Incorrect. Admin3 has the Owner role at the subscription level, which includes permission to assign roles.
Admin2 can create a resource group in the subscription: Yes – Incorrect. Admin2 has no role assignment, so they lack the necessary permissions to create resources.
