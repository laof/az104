Question : 41 Topic: 2

DRAG DROP -

You have an Azure Active Directory (Azure AD) tenant that has the contoso.onmicrosoft.com domain name.
You have a domain name of contoso.com registered at a third-party registrar.
You need to ensure that you can create Azure AD users that have names containing a suffix of @contoso.com.
Which three actions should you perform in sequence? To answer, move the appropriate actions from the list of actions to the answer area and arrange them in the correct order.
Select and Place:

Suggested Answer:

Question 2-41

Correct Answer (in order):
1. Add a custom name – You start by adding contoso.com as a custom domain name in Azure AD.
2. Add a record to the public contoso.com DNS zone – To prove ownership of the domain, you must add a TXT record to the DNS zone at your domain registrar.
3. Verify the domain – Once the DNS record is in place, you can verify the domain in Azure AD to complete the setup.
-----------------
Explanation of Other Options:
Add an Azure AD tenant – Incorrect. You already have an Azure AD tenant (contoso.onmicrosoft.com), so this step is unnecessary.
Configure company branding – Incorrect. This is optional and unrelated to enabling user creation with a custom domain.
Create an Azure DNS zone – Incorrect. The domain is registered with a third-party registrar, so DNS changes must be made there, not in Azure DNS.
