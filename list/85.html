HOTSPOT -

You have an Azure subscription named Subscription1 that contains a virtual network VNet1.
You add the users in the following table.
Which user can perform each configuration? To answer, select the appropriate options in the answer area.
NOTE: Each correct selection is worth one point.
Hot Area:

Suggested Answer:

Correct Answers:

Add a subnet to VNet1: User1 and User3 only
Assign a user the Reader role to VNet1: User1 only
-----------------

Explanation of Correct Options:

User1 and User3 only: User1 is the Owner and has full permissions, including adding subnets. User3 is a Network Contributor and has permissions to manage network resources like subnets. Therefore, both can add a subnet to VNet1.
User1 only: User1 as Owner has full access, including the ability to assign roles such as Reader to other users.
-----------------

Explanation of Other Options:

User3 only (for subnet): While User3 can add subnets, this option excludes User1, who also has the necessary permissions.
User2 and User3 only (for subnet): User2 is a Security Admin and does not have permissions to manage network resources like subnets.
User1, User2, and User3 (for subnet): User2 lacks the required permissions, so this option is incorrect.
User2 only (for assigning Reader role): User2 cannot assign roles as a Security Admin.
User3 only (for assigning Reader role): User3 as Network Contributor cannot assign roles.
Users 1 and 2 only (for assigning Reader role): User2 lacks role assignment permissions.
Users 2 and 3 only (for assigning Reader role): Neither User2 nor User3 can assign roles.
