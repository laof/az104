Question : 47 Topic: 2

You have an Azure Active Directory (Azure AD) tenant.

You plan to delete multiple users by using Bulk delete in the Azure Active Directory admin center.
You need to create and upload a file for the bulk delete.
Which user attributes should you include in the file?

  • A..The user principal name and usage location of each user only
  • CorrectB..The user principal name of each user only
  • C..The display name of each user only
  • D..The display name and usage location of each user only
  • E..The display name and user principal name of each user only
Suggested Answer:

Correct Answer:

B..The user principal name of each user only
-----------------

Explanation of Correct Option:

The user principal name (UPN) is the unique identifier for a user in Azure Active Directory. When performing a bulk delete operation, Azure AD requires only the UPN to identify and delete the user accounts. Other attributes like display name or usage location are not necessary for this operation.

Explanation of Other Options:

A..The user principal name and usage location of each user only — Incorrect because usage location is not required for deletion; only the UPN is needed.
C..The display name of each user only — Incorrect because display names are not unique and cannot reliably identify users for deletion.
D..The display name and usage location of each user only — Incorrect for the same reasons as above; neither attribute is sufficient or required for deletion.
E..The display name and user principal name of each user only — Incorrect because while UPN is required, display name is unnecessary and not used in the bulk delete process.
