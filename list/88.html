Question : 48 Topic: 2

HOTSPOT -

You have an Azure subscription named Sub1 that contains the Azure resources shown in the following table.
You assign an Azure policy that has the following settings:
✑ Scope: Sub1
✑ Exclusions: Sub1/RG1/VNET1
✑ Policy definition: Append a tag and its value to resources
✑ Policy enforcement: Enabled
✑ Tag name: Tag4
✑ Tag value: value4
You assign tags to the resources as shown in the following table.
For each of the following statements, select Yes if the statement is true. Otherwise, select No.
NOTE: Each correct selection is worth one point.
Hot Area:

Suggested Answer:

Correct Answers:

RG1 has the Tag2:IT tag assigned only. — Yes
storage1 has the Tag1:subscription, Tag2:IT, Tag3:value1, and Tag4:value4 tags assigned. — No
VNET1 has the Tag2:IT and Tag3:value2 tags assigned only. — No
-----------------

Explanation of Correct Option:

RG1 has only Tag2:IT directly assigned. No policy applies additional tags to RG1, and Azure does not inherit tags from parent resources by default.
-----------------

Explanation of Incorrect Options:

storage1 has Tag3:value1 directly assigned. Although it is within the scope of the policy that appends Tag4:value4, the policy only affects new or modified resources. Since there's no indication that storage1 was created or modified after the policy was applied or that remediation was run, it would not have Tag4:value4. Also, Azure does not automatically inherit Tag1:subscription or Tag2:IT from parent resources.
VNET1 has Tag3:value2 directly assigned. It is explicitly excluded from the policy, so it does not receive Tag4:value4. Like storage1, it does not inherit Tag2:IT from RG1 unless a separate inheritance policy is in place, which is not mentioned.
