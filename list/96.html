Question : 56 Topic: 2

You have 15 Azure subscriptions.

You have an Azure Active Directory (Azure AD) tenant that contains a security group named Group1.
You plan to purchase additional Azure subscription.
You need to ensure that Group1 can manage role assignments for the existing subscriptions and the planned subscriptions. The solution must meet the following requirements:
✑ Use the principle of least privilege.
✑ Minimize administrative effort.
What should you do?

  • A..Assign Group1 the Owner role for the root management group.
  • CorrectB..Assign Group1 the User Access Administrator role for the root management group.
  • C..Create a new management group and assign Group1 the User Access Administrator role for the group.
  • D..Create a new management group and assign Group1 the Owner role for the group.
Suggested Answer:

Correct Answer:

B.. Assign Group1 the User Access Administrator role for the root management group.
Explanation: Assigning the User Access Administrator role at the root management group level allows Group1 to manage role assignments across all current and future subscriptions under that root. This approach follows the principle of least privilege because it grants only the necessary permissions to manage access, not full resource control, and it minimizes administrative effort by applying the role at the highest level.
-----------------

Other Options:

A.. Assign Group1 the Owner role for the root management group.
This grants excessive permissions beyond role assignment, violating the principle of least privilege.
C.. Create a new management group and assign Group1 the User Access Administrator role for the group.
This limits the scope to only the new group, not covering existing subscriptions unless they are moved, which increases administrative effort.
D.. Create a new management group and assign Group1 the Owner role for the group.
Similar to option C, this does not cover existing subscriptions and also grants more permissions than necessary.
