You have an Azure subscription that contains the following users in an Azure Active Directory tenant named contoso.onmicrosoft.com:
User1 creates a new Azure Active Directory tenant named external.contoso.onmicrosoft.com.
You need to create new user accounts in external.contoso.onmicrosoft.com.
Solution: You instruct User2 to create the user accounts.
Does that meet the goal?

  • A..Yes
  • CorrectB..No
Suggested Answer:

Correct Answer:

B.. No
Explanation: User2 is a Global Administrator in the original tenant (contoso.onmicrosoft.com), but that role does not automatically apply to the new tenant (external.contoso.onmicrosoft.com). Each Azure AD tenant is isolated, and roles must be explicitly assigned per tenant. Therefore, unless User2 is also assigned a role like Global Administrator in the new tenant, they cannot create user accounts there.
-----------------

Other Option:

A.. Yes
This is incorrect because being a Global Administrator in one tenant does not grant permissions in another tenant. User2 would need to be assigned the appropriate role in the external.contoso.onmicrosoft.com tenant to perform user management tasks.
