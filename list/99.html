You have an Azure subscription that contains the following users in an Azure Active Directory tenant named contoso.onmicrosoft.com:
User1 creates a new Azure Active Directory tenant named external.contoso.onmicrosoft.com.
You need to create new user accounts in external.contoso.onmicrosoft.com.
Solution: You instruct User4 to create the user accounts.
Does that meet the goal?

  • A..Yes
  • CorrectB..No
Suggested Answer:

Correct Answer:

>B.. No
Explanation: User4 is an Owner at the Azure Subscription level, which grants permissions to manage resources within that subscription, but not within Azure Active Directory. Creating user accounts is a directory-level task, which requires roles like Global Administrator or User Administrator in the specific Azure AD tenant. Since User4 has no role in the new tenant (external.contoso.onmicrosoft.com), they cannot create user accounts there.
-----------------

Other Option:

A.. Yes
This is incorrect because subscription-level roles like Owner do not grant permissions to manage Azure AD users. Directory-level roles are required for user creation in Azure AD.
